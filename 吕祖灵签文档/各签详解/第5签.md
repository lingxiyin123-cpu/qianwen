# 第5签

**第5签详解**

**签文**：
```
柔顺而静，坤之六爻皆吉；
阴盛于阳，不怕亢龙之悔；
若此消息，只待羊兔相宜。
秋禾春麦应时收，勤种勤耕乃有期；
```

**详解**：

吕祖灵签第5签为中吉签，但需结合行事态度判断吉凶。此签以“古人亢龙得水”为典故，强调“柔顺守静、勤勉谨慎”的核心思想，具体分析如下：

签文：
柔顺而静，坤之六爻皆吉；
阴盛于阳，不怕亢龙之悔；
若此消息，只待羊兔相宜。

附诗：
秋禾春麦应时收，勤种勤耕乃有期；
不料惰农期逸获，一无敛获始知迟。

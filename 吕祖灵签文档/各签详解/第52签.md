# 第52签

**第52签详解**

**签文**：
```
虎坐中堂，鸡太相闹；
天不如人求，反增多少笑。
安而不危，菊花天到。
天道本无私，良心各自持；
```

**详解**：

吕祖灵签第52签为中平签，签文主题为“古人仁贵父子相会”，通过薛仁贵误射其子丁山的故事，强调谨慎行事、避免急躁的重要性。以下是综合分析：

签诗
虎坐中堂，鸡太相闹；
天不如人求，反增多少笑。
安而不危，菊花天到。

附诗
天道本无私，良心各自持；
倘然全昧却，后况更难知。

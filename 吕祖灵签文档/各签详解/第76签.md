# 第76签

**第76签详解**

**签文**：
```
「水落石出，荣华已矣！若得一通，除非再起；不宜前进，当守时势。」
不如人处听升沉，安份常存退处心；
君子劳谦终获吉，有时瓦砾变黄金。
卦象：命运不通，吉中带凶，轻躁冒进则终局不利。
```

**详解**：

吕祖灵签第76签为中下签，主题为「古人六出祈山」，借三国时期诸葛亮六次北伐未果的历史典故，警示求签者运势未通时需谨慎守成。以下是综合分析：

「水落石出，荣华已矣！若得一通，除非再起；不宜前进，当守时势。」
诗曰：
不如人处听升沉，安份常存退处心；
君子劳谦终获吉，有时瓦砾变黄金。

此签主「运势未通，吉中藏凶」，强调「守成」而非冒进。

卦象：命运不通，吉中带凶，轻躁冒进则终局不利。
启示：人生进退有常，得志时与众人共进，失意时独善其身，方为长久之道。

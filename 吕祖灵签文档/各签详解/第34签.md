# 第34签

**第34签详解**

**签文**：
```
吕祖灵签第34签为“古人龙马负图”，综合多个来源分析如下：
自从河洛周天数，演到出门交有功；
一生际遇须龙马，不用踌躇自合行。
一二三四五六七，八九相乘数乃毕；
```

**详解**：

吕祖灵签第34签为“古人龙马负图”，综合多个来源分析如下：

签诗：
自从河洛周天数，演到出门交有功；
一生际遇须龙马，不用踌躇自合行。

诗曰：
一二三四五六七，八九相乘数乃毕；
老阳未变不能全，占者逢之静则吉。

源于伏羲时期“龙马负河图”传说，象征天赐良机。卦象为“运至时来”，需主动把握但忌冒进，阴阳未全时宜静待。

# 第20签

**第20签详解**

**签文**：
```
吕祖灵签第20签为「古人王道人炼丹」，属中吉签类，主要强调坚持与渐进的重要性。以下是综合分析：
九转丹成白日升，神仙有数利人营；
一朝走马龙行会，此是荣华一美因。
八九玄功已有基，频添火候莫差迟。
```

**详解**：

吕祖灵签第20签为「古人王道人炼丹」，属中吉签类，主要强调坚持与渐进的重要性。以下是综合分析：

主签文（综合多个版本）：
九转丹成白日升，神仙有数利人营；
一朝走马龙行会，此是荣华一美因。

辅助诗曰：
八九玄功已有基，频添火候莫差迟。
待看十二重楼透，便是丹成鹤到时。

# 第51签

**第51签详解**

**签文**：
```
镜破月缺，孙山保得；
若要安康，酬神免业。
他乡孤客迍邅无聊之象，求谋不惟无济，且恐破财惹祸，不如忍耐修德，善自排遣。
云横山际水茫茫，千里长途望故乡；
```

签文：
镜破月缺，孙山保得；
若要安康，酬神免业。

# 第36签

**第36签详解**

**签文**：
```
芹香鹿鸣，不远不近；二五之内，自然有幸，天喜文昌照命。
檐前喜鹊噪，房中灯芯报；忧愁渐渐消，诸事咸称好。
此签以唐代名臣张九龄为典故。他出生时鹿鸣于野、紫气东来，象征“文昌星照命”，预示才华与仕途显达。
```

**详解**：

吕祖灵签第36签为上吉签，主题为“古人张九龄出世”，象征紫气东来、文昌照命，预示好运与成就。以下是综合分析：

签文：
芹香鹿鸣，不远不近；二五之内，自然有幸，天喜文昌照命。
诗曰：
檐前喜鹊噪，房中灯芯报；忧愁渐渐消，诸事咸称好。

此签以唐代名臣张九龄为典故。他出生时鹿鸣于野、紫气东来，象征“文昌星照命”，预示才华与仕途显达。

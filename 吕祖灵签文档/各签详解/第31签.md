# 第31签

**第31签详解**

**签文**：
```
水上顺风吹，高才得意挥；
学成文武艺，自有好栽培。
出处遭奸妒，山穷路转迷；
水急舟难渡，万事莫强求。
```

**详解**：

吕祖灵签第31签为中平签，签文以三国时期姜维辅佐蜀汉的历史典故为背景，强调隐忍、谨慎和行善的重要性。以下是综合分析：

签词

水上顺风吹，高才得意挥；
学成文武艺，自有好栽培。

诗曰
出处遭奸妒，山穷路转迷；
水急舟难渡，万事莫强求。

此签虽非大凶，但强调“万事莫强求”。求签者需以姜维为鉴，在顺境中保持清醒，逆境中修德向善，方可转危为安。若需完整签文来源，可参考《乾坤网》《天气网》等解签记录。

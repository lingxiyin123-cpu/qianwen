# 第94签

**第94签详解**

**签文**：
```
事已如此，悔之何及；
纵有不祥，还宜阴骘。
愁面笑颜开，忧心事可谐；
但凭依理做，不必费疑猜。
```

**详解**：

吕祖灵签第94签为“古人蒙正捞斋”，属先苦后甜、先难后易之象，整体吉凶可判为中吉，但需结合自身修德与坚持方能应验。以下为综合分析：

签词：
事已如此，悔之何及；
纵有不祥，还宜阴骘。

此签以北宋名臣吕蒙正的典故为背景。吕蒙正少时家贫，寄居寺院苦读，却遭僧人轻视，故意饭后敲钟使其无斋可食。他忍辱勤学，终中状元，成就功名。

核心寓意：

诗曰：
愁面笑颜开，忧心事可谐；
但凭依理做，不必费疑猜。

卦象：
先苦后甜，需修德行仁，福报在后。

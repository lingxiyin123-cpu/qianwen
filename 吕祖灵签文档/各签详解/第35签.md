# 第35签

**第35签详解**

**签文**：
```
吕祖灵签第35签为上吉签，结合《白马跳檀溪》典故与签文解析，具体分析如下：
阴德不是非凡事，积善之家庆有余；
厄去福来天凑巧，龙飞虎跃一鸡啼。
门内起干戈，亲仇两不和；
```

**详解**：

吕祖灵签第35签为上吉签，结合《白马跳檀溪》典故与签文解析，具体分析如下：

主签文
阴德不是非凡事，积善之家庆有余；
厄去福来天凑巧，龙飞虎跃一鸡啼。

附诗
门内起干戈，亲仇两不和；
朱衣临日月，始觉笑呵呵。

此签源自《三国演义》中刘备骑“的卢马”跃檀溪脱险的故事。寓意绝境中凭借德行与机缘逢凶化吉，呼应签文“厄去福来天凑巧”。

如需查看完整签文来源，可参考、2、5、6（万年历、大家找等平台）。

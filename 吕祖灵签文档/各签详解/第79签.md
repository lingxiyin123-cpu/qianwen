# 第79签

**第79签详解**

**签文**：
```
不是知音不与论，只看镜破风不正；
逆水莫行舟，气散宜当尽。
凡事莫欺心，欺心祸即临；
守遇安我拙，或可免灾侵。
```

**详解**：

吕祖灵签第79签为中下签，整体预示破镜难圆、运势多阻，需谨守正道以避灾祸。以下为综合分析：

签文
不是知音不与论，只看镜破风不正；
逆水莫行舟，气散宜当尽。

诗曰
凡事莫欺心，欺心祸即临；
守遇安我拙，或可免灾侵。

此签核心警示“破镜难圆”，主运势低迷、家庭情感动荡，但若能安守本分、以诚待人，可减轻灾厄。需注意：不同解签版本对吉凶程度略有差异，部分提及“中平”（如），但综合多数分析，仍以中下签为主调。

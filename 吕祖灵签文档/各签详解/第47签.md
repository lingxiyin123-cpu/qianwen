# 第47签

**第47签详解**

**签文**：
```
已成了，又难保，只待舟济顺风行跑；
每事多烦恼，虔心神前修省长安保。
万里从军仗剑行，兔孤巢穴未能平；
老不封候嗟已晚，夜阑灯火独伤神。
```

**详解**：

吕祖灵签第47签为中平签，以三国时期曹洪失潼关的历史典故为背景，强调戒骄戒躁、顺应时势的重要性。以下是具体分析：

签文：
已成了，又难保，只待舟济顺风行跑；
每事多烦恼，虔心神前修省长安保。

诗曰：
万里从军仗剑行，兔孤巢穴未能平；
老不封候嗟已晚，夜阑灯火独伤神。

此签核心在于“守成”与“谦逊”。需以曹洪为鉴，在顺境中保持清醒，逆境中调整心态。行事遵循“虔心修省，顺应天时”，方能趋吉避凶。若需更完整签文背景，可查阅《三国志》或道教签书原文。

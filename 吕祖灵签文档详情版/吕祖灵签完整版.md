## 第1签

**第1签：古人王母祝寿**

**签文**：
```
蓬莱东阁玉桃香，顺水行舟仙赐方；
宜男正好图全计，不必他方卜地长。
```

**详解**：

**签文**：
蓬莱东阁玉桃香，顺水行舟仙赐方；
宜男正好图全计，不必他方卜地长。

**解曰**：
王母娘娘大寿，各仙家前往致贺，场面极爲浩大。王母对贺客皆赐以仙桃仙丹等长生不老的美食。衆仙大会，乐韵钧天，属吉庆徵兆。

**详解**：
谋望
不要急于成事，须待时机。
钱财
委託异性去处理，收穫较易较佳。
婚姻
虽吉，但仍要守候。
自身
修善即可平安。
家宅
家宅平安，逐渐昌盛。
开业
与朋友合资易有所成。
迁居
选吉方即可。
出行
往远方，有意外之喜。
疾病
一切小病痛都能痊愈。
六甲
顺产，生男孩。
行人
音信将至。
诉讼
双方可得和解。

**又曰**：
占得鳌头百事成，逢迎到处不须疑；
从兹修省能方便，福禄绵绵自可期。

**释义**：
在运势尚未通达的时候，当然是有志难伸，一旦鳌头独佔。跟红顶白、锦上添花的人会接踵而来，如果一下子利欲薰心，很容易把大好前程新自断送。这段时候最 好是自我签讨，保持稳健踏实的态度，才能够使到福禄绵绵不绝。

## 第2签

**第2签：古人潜龙变化**

**签文**：
```
潜藏自有光明日，守耐无如待丙丁；
龙虎相争生定数，春风一转渐飞惊。
```

**详解**：

**签文**：
潜藏自有光明日，守耐无如待丙丁；
龙虎相争生定数，春风一转渐飞惊。

**解曰**：
古人以龙喻示人的际遇，潜龙即隐藏着的龙，爲什么要隐藏，因爲时运不际，未得其时，所以潜藏以待，一旦风云际会，即可飞龙在天，利见大人了。 求得此签者，万事待时，自有光明之日。至于腾飞之时，当在丙丁月日。

**详解**：
谋望
暂仍未至时机，待丙丁月日。
钱财
委託他人协助，可得成功。
婚姻
目前易有磨擦及难以解决的麻烦，只要耐心地去处理，是会美满的。
自身
处于被子困局面，一进一退都要小心谨慎，渡过暑天，事事转吉。
家宅
是是非非相当繁多，不要被这些口舌所困扰。
开业
初期稍多阻滞，日后业绩大有所爲。
迁居
选吉方则宜。
出行
不宜远行，易多烦恼。
疾病
被寒热所困，长久会有危险。
六甲
应多休息，勤于检查身体，利女孩。
行人
远方的人较迟至。
<p>
	诉讼
 一时难以解决。

**又曰**：
贵人相逢更可期，庭前枯木凤来仪；
好将短事从长事，休听旁人说是非。

**释义**：
人有三衰六旺，衰的时间短，旺的时间长；旺的时间有贵人相逢，有所扶助，连门庭枯木都有凤凰来朝。要保持有这种吉祥之运，最重要的是要本着忠信孝悌，尤 其是不要胡乱听从旁人的口舌是非，打乱自己一往秉持的信念。

## 第3签

**第3签：古人唐明皇游月殿**

**签文**：
```
明月当三五，天地自无私；
一阳来已複，得意在秋时。
```

**详解**：

**签文**：
明月当三五，天地自无私；
一阳来已複，得意在秋时。

**解曰**：
唐明皇梦游于太虚之间，欲一会嫦娥，心念所至，游至月殿，得睹嫦娥歌舞，即霓裳羽衣舞，仙音仙乐，遂得以传至凡间。求得此签者，当知月在十五，爲月圆之夜，阴盛阳衰，喻示小人得志，凡事不可相争，以忍气爲妙；此签若在秋季时求得更爲大吉。

**详解**：
谋望
八九月时可成。
钱财
委託女性协助，财程顺遂。
婚姻
利恋爱，得美妻，但易有一种虚幻，不知所措，患得患失的心情。
自身
身体健康，却神不守舍，应该集中精神，方可吉利。
家宅
平安，有置业的欲望，在秋季时求得，大吉。
开业
吉，宜从事珠宝、美容、服装等修饰的业务。
迁居
远近都不适合。
出行
远方会有意外之喜。
疾病
易多情绪病，如讳疾忌医，病情会有恶化。
六甲
顺産，生女孩。
行人
未至。
诉讼
名正言顺地谈判，可得和解。

**又曰**：
牡丹富贵足称王，极盛遇虞未久长；
凡事必须留后步，与奢宁俭乃爲藏。

**释义**：
牡丹花被喻爲富贵之花，是衆花之魁首。在其极盛之时，时间很短促。但只要在做任何事情之时，留着后步（不要去得太尽），这些就不会有什么凶险了。占者防小人得志而对己不利。

## 第4签

**第4签：古人唐三藏取经**

**签文**：
```
正大有鬼神之助，吉祥成忠厚之报；
不怕邪魔小崇，只看秋收冬藏。
```

**详解**：

**签文**：
正大有鬼神之助，吉祥成忠厚之报；
不怕邪魔小崇，只看秋收冬藏。

**解曰**：
唐时唐三藏立志弘扬佛法，爲寻求正道，往西天竺取经，曆尽千辛万苦，涉足山川险阻，凭着坚持之心，鬼神之庇护，终于得偿所愿，满载而归。求得此签者，只要行事光明正大，忠厚待人，必有鬼神之助，逢凶化吉。求谋以秋季爲有利。

**详解**：
谋望
待到秋季时节有成。
钱财
春夏破财，秋冬有盈馀。
婚姻
暂时仍未到时机。
自身
坚心守志以修身。
家宅
行忠厚合家平安。
开业
选择秋冬季爲吉。
迁居
远方较合适。
出行
远处有贵人相助。
疾病
略有凶险，无碍。
六甲
生産顺利。
行人
已在途中。
诉讼
虽胜也要费很多时日。

**又曰**：
人心叵测却难防，幸有神明鑑在旁；
若使坚持心不改，一生偃蹇理应当。

**释义**：
人世间最难忖测的是人心，所谓「心猿意马」，一刹那间人的思维可以有很大的变化和波动，因此爲善爲恶也在一念之间。若能使自己时刻秉持忠厚、正大之心，则一生不会经历大凶险，亦不会有太大的蹇滞出现。

## 第5签

**第5签：古人亢龙得水**

**签文**：
```
柔顺而静，坤之六爻皆吉。
阴盛于阳，不怕亢龙之悔。
若此消息，只待羊兔相宜。
```

**详解**：

**签文**：
柔顺而静，坤之六爻皆吉。
阴盛于阳，不怕亢龙之悔。
若此消息，只待羊兔相宜。

**解曰**：
所谓亢龙，喻示人到极度好的时候；极度好是一件好事吗？严格来说，是不太好的，因爲是到了极限的缘故，由最高极限，不会再有更高的，接着来的只有是慢慢走下坡，因此必须要有抑压疏导。求得此签者，凡事小心则吉，最好能随遇而安。

**详解**：
谋望
随遇而安，不可强求。
钱财
一有差错即破财。
婚姻
是男方在拖延。
自身
凡事小心以修身。
家宅
勤奋可兴家。
开业
怠惰即可败业。
迁居
近处无妨。
出行
远行寻友恐不遇。
疾病
立即就医可愈。
六甲
顺産，生女孩。
行人
卯日、未日可至。
诉讼
对己方不利，以和解爲宜。

**又曰**：
秋禾春麦应时收，勤种勤耕乃有期；
不料惰农期逸获，一无敛获始知迟。

**释义**：
田园作业，四时有序。播种、插秧、收割、打禾都有一定时节。农人只要按时勤耕当然有收穫，若贪图逸乐或一暴十寒，当然没有收成，到时悔恨亦已迟了。占者宜勤不宜惰，凡事小心则吉。

## 第6签

**第6签：古人刘备取西川**

**签文**：
```
如争如战，长子克家，从吉而贞。
当知天从人愿，不必二二三三。
```

**详解**：

**签文**：
如争如战，长子克家，从吉而贞。
当知天从人愿，不必二二三三。

**解曰**：
三国初期，刘备仍未建立蜀国，无立足之地，故进取西川，以爲安身立命之地。得诸葛孔明之助力，终夺取西川。求得此签者，营谋合宜，且必定得到，是属于平步青云之象。但须谨慎，如欲以奸以营谋他人，则此签凶险非常。

**详解**：
谋望
心宽志广则可成功。
钱财
金钱运大吉，不利横财。
婚姻
仍未到时机。
自身
不妒忌人以修身。
家宅
家和万事兴。
开业
平步踏上青云之路。
迁居
远处须待，近处吉。
出行
近处公干吉。
疾病
久病恐有危机。
六甲
生産十分顺利。
行人
将至，音信先到。
诉讼
己方胜，但须迟些。

**又曰**：
山高水又长，福泽本汪洋；
方寸修爲好，祖德宗功扬；
龙呤云雨至，冲霄上九重。

**释义**：
祖宗积德行善，福泽被及子孙，如果子孙又能恪守祖训，以仁德行事，其人将不虞受凶险所侵。更进一步来说，如果能遇到适当时机，则名声可直上九重天，爲人所仰望，这些天从人愿的事，就是从积福德所致。

## 第7签

**第7签：古人盘古初开**

**签文**：
```
临物易与，天地人合；
一到春风，百祥骈集。
```

**详解**：

**签文**：
临物易与，天地人合；
一到春风，百祥骈集。

**解曰**：
天地溷沌，盘古初开，万物未齐，一切皆由天地慢慢地孕育出来。由于阴阳的涵盖，三才俱备，一到春风化育，万物祯祥。求得此签者，如能应时，则百事遂心，是大吉之兆，如春风未到，则困在这段时间，往后才能名利双收。

**详解**：
谋望
时机未到，半途恐有阻滞。
钱财
有障碍，难望急成。
婚姻
初婚不宜，再婚宜。
自身
修善平安可发达。
家宅
春风一至，百事吉祥。
开业
初段阻滞，慢慢转好。
迁居
延迟爲佳。
出行
小心选择时机。
疾病
治癒后不可不注意疗养。
六甲
宜十分当心。
行人
过几日有消息至。
诉讼
急则恐防败诉。

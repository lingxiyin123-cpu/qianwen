# 第81签

**第81签：古人合兵破曹火烧赤壁**

**签文**：
```
合而分，分而合；
天时人事，雨斟酌。
```

**详解**：

**签文**：
合而分，分而合；
天时人事，雨斟酌。

**解曰**：
三国时代，魏、蜀、吴三分天下，各自都希望打败对方，一统天下。魏国曹操率领百万大军南下，欲灭蜀。时蜀主刘备兵微将寡，难以抗衡，遂联络吴主孙权共同抵御曹兵。藉着地利与人事的调动得宜，终于火烧赤壁，得以大败曹操，从而取得鼎足三立之局。求得此签者，作事吉，但亦须事事与人商讨后才做，不可独断独行，这样凡事皆吉，可以无恙。

**详解**：
谋望
需有人助始能成功。
钱财
财务周转上无困难。
婚姻
再认识深一点才论婚嫁。
自身
劳心始得平安。
家宅
家运安然。
开业
与人合资尚有可爲。
迁居
在附近地区宜，远则不宜。
出行
外出公干有贵人助。
疾病
速治易痊愈，迟恐病情有变。
六甲
会生子。
行人
将至，有好消息。
诉讼
和解对双方有利，会很快平息。

**又曰**：
造物弄人多鹘突，纷纷成败难稽核；
无心插柳柳成荫，有意栽花花不发。

**释义**：
造物者对人的命运有时也是每多作弄的，因此要计算自己的成功失败亦是没有甚么可根的。在这样情况下，往往有心栽花的，未必能长满花朵，而无心插柳的，偏偏它就很茂盛。因此一得一失，一成一败都有前定之数，凡事与他人合作都会成功，但亦须防合久则分。

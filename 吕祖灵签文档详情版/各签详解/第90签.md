# 第90签

**第90签：古人生祭李彦贵**

**签文**：
```
雨打梨花，人在天涯，
功名多费力， 图事不能成，
阴阳反复，时过令人嗟。
```

**详解**：

**签文**：
雨打梨花，人在天涯，
功名多费力， 图事不能成，
阴阳反复，时过令人嗟。

**解曰**：
李彦贵求功名自家乡上京赴考，途中遇难获救，家人未悉，以爲已赴险曹，故祭之。求得此签者，运蹇时乖，诸事欠吉，加上事事反反覆覆，时好时坏 ，令人摸不着头脑。唯有自已处处小心，靠神灵的庇佑，朋友的协助，然后才能从危机中得救援，一切须待时机。

**详解**：
谋望
有人辅助则可成。
钱财
托人筹措，可以成功。
婚姻
由尊长决定则吉。
自身
修业得平安。
家宅
宅未得宁。
开业
先改目前境况，再考虑其他。
迁居
无妨，但需与家人商量。
出行
宜早归。
疾病
要慢慢才能痊愈。
六甲
提防有意外。
行人
有阻障，不能来。
诉讼
有人阻挠，一时难以解决。

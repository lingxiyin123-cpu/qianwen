# 第74签

**第74签：古人仁贵投军**

**签文**：
```
勐虎噬人，恶犬欺生；
一箇昏天，如何扬眉吐气。
```

**详解**：

**签文**：
勐虎噬人，恶犬欺生；
一箇昏天，如何扬眉吐气。

**解曰**：
薛仁贵自少臂力惊人，武艺超卓，但生活清贫，于是报名投军，希望能从武事出身。虽然仁贵在军中屡立战功，但爲主帅冒名所夺，终至劳而无功。求得此签者，凡事辛苦，同时受小人羁绊太重，一切皆难开展。逢寅辰年、月则吉，此时开始转运。

**详解**：
谋望
劳而无功，但逢寅辰乃吉。
钱财
金钱暂未成就。
婚姻
姻缘不谐。
自身
由苦遂渐得甘。
家宅
宅运未开。
开业
初时艰辛。
迁居
稍迟有利。
出行
到底无妨。
疾病
缠绵床第。
六甲
安産，可生男。
行人
在原地未动。
诉讼
宜解，唯未至时机。

**又曰**：
又是君家命运低，如何屡次失东西；
劝君及早牢关锁，检点些儿也不迟。

**释义**：
时来运未至之际，作事如望梅止渴、画饼充饥，始终都是水中月、镜中花，是没有结果的。同时，更多的小人羁伴在侧，使你举步维艰，愈是想前进，愈受阻延。因此，应不畏辛苦做事，步步爲营，须防小人作祟，生出意外。

# 第25签

**第25签：古人冯诚修告老**

**签文**：
```
猪鼠相合，卯酉相同；
一船满载，载载不空，
得心应手；只在三四五成功。
```

**详解**：

**签文**：
猪鼠相合，卯酉相同；
一船满载，载载不空，
得心应手；只在三四五成功。

**解曰**：
冯诚修忠心爲国，以天之任爲己任，可惜奸臣当道，壮志不得伸展，遂不欲爲官，辞官归故里，以待其时。求得此签者，要懂得时机知所进退，并以退守爲最妙，故此勿心高气傲，一味冒进，适可而止，才可以见到吉祥。

**详解**：
谋望
以退爲进，可得遂意。
钱财
可取得不错的收益。
婚姻
是美满的婚姻。
自身
戒除心高气傲以修身。
家宅
莫强求合家平安。
开业
初段宜守，后方见吉祥。
迁居
远处方吉。
出行
非公干则无碍。
疾病
急病，慢病都无妨，可愈。
六甲
顺産，可得男孩。
行人
已在途中。
诉讼
要儘快解决，否则败诉。

**又曰**：
寅午戍年多阻滞，亥子丑月渐亨嘉；
更逢玉兔金鸡会，枯木逢春自放花。

**释义**：
寅午戍合成火局，因此在属火之年会很多阻滞，亥子丑合成水局，因此在属水之月会渐渐享通。故占得此签的人，在火年遇木（卯、玉兔）月，如果有金（酉、金鸡）来克制木的话，则可以使不成火局。故此卦是喜水而忌火。

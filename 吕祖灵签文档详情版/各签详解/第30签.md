# 第30签

**第30签：古人湘子得道**

**签文**：
```
乾三爻，渐三爻；
一时遇合，鹤鸣九皋。
```

**详解**：

**签文**：
乾三爻，渐三爻；
一时遇合，鹤鸣九皋。

**解曰**：
湘子即是韩湘子，爲传说中的八仙之一。传爲韩愈族侄，后得道成仙。以易卦而言，一九以成十数，而六与十相逢谓之遇合，此时即可转爲昌盛，正如鹤鸣于九天之中，任你遨游。求得此签者，则运颇爲亨通，但名位则仍未彰显，凡作事如逢六或十则有好佳音。

**详解**：
谋望
有成就之兆。
钱财
可以垂手取得。
婚姻
暂未至时机。
自身
心地光明以修身。
家宅
家庭和睦则安乐自在。
开业
忍耐初段便可获利。
迁居
无论何处均可。
出行
途中会遇贵人。
疾病
病发即延医，可迅速治癒。
六甲
顺産，多爲生女孩。
行人
六、十之日可至。
诉讼
谈判若急躁便会吃亏。

**又曰**：
晓雨初晴映碧溪，重重春色上柴扉；
黄金不尽家殷富，何必区区羡锦衣。

**释义**：
人生追求不外乎名利，有些人名利双收，有些人名利俱无，有此人有名无利，有此人是有利无名。无论是属于那一种，只要知足便能生活无忧。「知足者，贫亦乐；不足知者，富亦忧。」此喻示家境富足无名禄，劝人不必因此而羡慕名位的人。

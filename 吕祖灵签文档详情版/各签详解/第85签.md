# 第85签

**第85签：古人伦文叙不第**

**签文**：
```
一去中阻，云雾不开；
且自宽怀，须许善愿无碍。
```

**详解**：

**签文**：
一去中阻，云雾不开；
且自宽怀，须许善愿无碍。

**解曰**：
伦文叙幼读经书，文思畅达，尤擅长于吟诗作对，常有惊人之语。其诗句用词显浅而多触景，故爲乡里所称颂，有「鬼才」之誉。他未发迹时，家境清贫，尝参加考试落第，后得中状元，光耀门楣。求得此签者，不管是求名抑或求利，皆有阻障。须经历千辛万苦，始得吉祥。

**详解**：
谋望
暂时仍未开运。
钱财
尚待时机。
婚姻
迟一点，终必会成功。
自身
洁身自爱当无碍。
家宅
积德便宽怀。
开业
过一段时间较好。
迁居
不宜，居现处爲好。
出行
中途会有阻滞。
疾病
须医治较长时间。
六甲
怀孕须防饮食卫生。
行人
人不来，有资讯至。
诉讼
早点解决损失较少。

**又曰**：
张琴待月，月到天中；
时不可失，触事成春。

**释义**：
在庭院里打开弦琴，以等待月亮，当月亮运行在天中时，光华映耀，庭院生辉，在此时抚弄琴弦，以作和应，当能触景而生情，但须有节制及忍耐才能达到这个目标。

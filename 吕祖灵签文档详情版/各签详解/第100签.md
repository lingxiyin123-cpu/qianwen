# 第100签

**第100签：古人孔明求寿**

**签文**：
```
月落星沉，风生雨至；
只要龙虎不相争，方可如时利。
```

**详解**：

**签文**：
月落星沉，风生雨至；
只要龙虎不相争，方可如时利。

**解曰**：
在三国时代，诸葛孔明爲蜀国之军师，自白帝城刘备临死托孤后，孔明矢言承继刘备之遗志，讨代魏国。兵至五丈原之际，孔明病重，于是在军营内设七七四十九盏灯长命灯，登坛求寿。可惜被蜀将魏延闯入，踢翻星灯，求寿不成。古人以星代表寿命，故「星沉」即寿终之意。求得此签者，宜修身自省，方可无百碍，而一切想进行之大小事宜，均有所不宜。 孔明求寿，却被魏延踢翻寿星灯，凡事不吉之兆。

**详解**：
谋望
不论大小，皆不成就。
钱财
未到时机。
婚姻
在重要关头告吹。
自身
宜修身自省，可无大碍。
家宅
恐家中有人患病。
开业
未曾开始已告倒闭。
迁居
急则有害，延期即可。
出行
路途多阻碍。
疾病
恐有丧服之厄。
六甲
注意健康。
行人
有阻碍，迟至。
诉讼
对方有利。

**又曰**：
桃李谢春风，西飞又复东；
家中无意绪，船在波涛中。

**释义**：
桃花、李树在春初发茂，盛放于春末，故春来是桃李由盛而衰之时节。而候鸟亦由迁居地飞返回原居地，人虽然是平静如常地，但外来的冲击颇大，如船置身于波涛之中。依此诗意，凡事均须小心谨慎，提防一切突然而来的变故。 
路不通，门闭塞，谨慎提防，云藏月黑，桃李谢春风，西飞又复东，家中无意绪，船在波涛中。

# 第67签

**第67签：古人张飞战张郃**

**签文**：
```
一跌复一跌，只恐卤莽灭裂；
果能惜得寸金，也要调摄。
```

**详解**：

**签文**：
一跌复一跌，只恐卤莽灭裂；
果能惜得寸金，也要调摄。

**解曰**：
张飞乃三国时蜀国一员勇将，兼有智谋，张郃是三国时魏国一名勐将，亦有智。于蜀魏交战期张飞遇张郃而开战，双方愈战愈勇，数次更换坐骑，由大家有虽强烈敌意以至惺惺相惜。求得此签者，只要待人忍让嫌和，当可趋吉避凶，和气致祥。

**详解**：
谋望
须得外人协助始成。
钱财
目前仍未有利润。
婚姻
有第三者阻挠未成。
自身
和气持身可呈祥。
家宅
家和万事兴。
开业
拍挡相和则兴业。
迁居
要谨慎考虑。
出行
仍未到适当时候。
疾病
缠绵床第。
六甲
安稳，多是生女。
行人
未动，有音讯至。
诉讼
滞蹇不要心急，忍耐有利。

**又曰**：
和气致祥，乖气致戾；
若要兴隆，和气爲计。

**释义**：
每个人的祸福都根由于因果，因果有来自先天命运，也有来自后天。以后天而言，和气待人，大家都不生争拗，自然可得祯祥。乖气凌人，大家互出计谋，自然是戾气冲天，所以要令自己得到兴盛的话，必须以和爲上上之策。故此忍让谦和，自然可趋吉避凶。

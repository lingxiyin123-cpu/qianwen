# 第47签

**第47签：古人曹洪失潼关**

**签文**：
```
已成了，又难保，只待舟济顺风行跑；
每事多烦恼，虔心神前修省长安保。
```

**详解**：

**签文**：
已成了，又难保，只待舟济顺风行跑；
每事多烦恼，虔心神前修省长安保。

**解曰**：
曹洪爲三国时曹操的堂弟，饶勇善战。曹操命洪守潼关以抵御蜀国刘备之大军，因自视过高，爲诸葛孔明设计破潼关。求得此签者，须戒暴躁，急进及自以爲是的性格，很容易被人所乘而致后悔莫及。故当安份守纪，以静爲吉。

**详解**：
谋望
难成。
钱财
中途生意外，周转不灵。
婚姻
暂时未能成就。
自身
安份守纪则安。
家宅
是非莫管爲宜。
开业
不适宜进行新计划。
迁居
仍未是时机。
出行
近处可无妨。
疾病
较长时间才能治癒。
六甲
安産，是女孩。
行人
阻滞在途中。
诉讼
宜儘快解决，迟则生变。

**又曰**：
万里从军仗剑行，兔孤巢穴未能平；
老不封候嗟已晚，夜阑灯火独伤神。

**释义**：
出外征战唯凭仗剑而行，可惜有时连很少的敌人都未必能够一下子消灭，到头来虽然是戎马半生，而且功劳不少，但仍未能得到封候拜将，徒然惹得自己伤悲。因此看来，是非閒事莫管，亦切勿妄求，而且一心向善，诸恶勿作，神必保佑吉昌。

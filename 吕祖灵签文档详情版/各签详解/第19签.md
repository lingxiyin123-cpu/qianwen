# 第19签

**第19签：古人崔梦熊得第**

**签文**：
```
喜得秋风万事足，一举成中天下福；
风凰五彩利飞行，只看子午时日卜。
```

**详解**：

**签文**：
喜得秋风万事足，一举成中天下福；
风凰五彩利飞行，只看子午时日卜。

**解曰**：
崔梦熊初时运蹇时乖，屡试不第。后再奋进，勤加苦读，终在秋围之试得第，扬名天下。求得此签者，喻示努力坚心，做事有始有终，终必会名利成就，春风得意，喜气盈门。时间当应在子午年月日。

**详解**：
谋望
可以成功，但要看是否坚持。
钱财
会有七八成把握。
婚姻
易成且美好。
自身
坚持有始有终则大吉。
家宅
渐趋昌盛。
开业
只要有始有终，当可成功。
迁居
与伴侣商量后即好。
出行
会比预期早返回。
疾病
小心脚部的病患。
六甲
顺産，多産女孩。
行人
子午日会到。
诉讼
千万不能性急地谈判。

**又曰**：
月老冰人说合成，红丝繫就不劳心。
百年伉丽成偕老，瓜瓞绵绵啓后昌。

**释义**：
月老即冰人，职司撮合人姻缘的仙人。只要他将红丝繫上，男女虽爲仇敌，或远在天涯，也会得到结合，完全不需要自己劳心劳神。在那段时间，只要能够坚守自己的岗位，则在农曆秋季时节，名利会有成就，兼且喜气盈门。

# 第35签

**第35签：古人白马跳檀溪**

**签文**：
```
阴德不是非凡事，积善之家度有馀；
厄去福来天凑巧，龙飞虎跃一鸡啼。
```

**详解**：

**签文**：
阴德不是非凡事，积善之家度有馀；
厄去福来天凑巧，龙飞虎跃一鸡啼。

**解曰**：
三国时蜀国刘备将降将坐骑「的卢」收爲己用。后赴刘表宴会，蔡瑁等人欲杀备，备潜出乘的虏逃亡。堕襄阳城西檀溪之中，不能出。备说：「的卢，今日厄矣，可努力！」的卢一跃三丈，遂得脱。得此签者，虽危而有救，险处逢生，凡事谨慎。

**详解**：
谋望
可成，唯须待时机。
钱财
可以获利。
婚姻
双方仍未有意结合。
自身
人情世故爲治身之本。
家宅
慎防同室操戈，兄弟不睦。
开业
须防小人阻挠。
迁居
远不吉，近可犹。
出行
有危，但有贵人扶助。
疾病
急病有危。
六甲
须小心调理初生婴儿身体。
行人
在途中，有阻滞。
诉讼
有凶险，须提防。

**又曰**：
门内起干戈，亲仇两不和；
朱衣临日月，始觉笑呵呵。

**释义**：
同室操戈，兄弟不和，是很多家庭中常有发生的事，究其原因，当然是当父母的处事不公平，日积月累而引起的。因此家不齐，身不修就不能当上大事，得到高职。要厄去福来，必须是积善之家，同时要时常行阴德之事才可以，这也是人情练达的一种表现，只要事事小心，遇厄也可无碍。

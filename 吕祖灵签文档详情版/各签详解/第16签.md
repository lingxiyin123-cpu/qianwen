# 第16签

**第16签：古人王勃中状元**

**签文**：
```
时来风送滕王阁，运至何忧跨仙鹤。
甲乙两运天云梯，也知桂香味早卓。
```

**详解**：

**签文**：
时来风送滕王阁，运至何忧跨仙鹤。
甲乙两运天云梯，也知桂香味早卓。

**解曰**：
王勃爲初唐四大才子之一，年纪轻轻便聪明绝顶，文思非常敏捷，其着名作品滕王阁序更传诵千古，爲不可多得之佳品。所以常人以「时来风送滕王阁」来表示时来运到，春风得意。求得此签者，事事如意，若能积德行仁，更会逢凶化吉。

**详解**：
谋望
营谋会事事如意。
钱财
绝对可以有收穫。
婚姻
相当美满。
自身
积德行仁以修身。
家宅
有锦上添花之喜。
开业
不须太费力便可昌盛。
迁居
任何地方都吉。
出行
无妨，并有意外之喜。
疾病
看来很严重，但很快治癒。
六甲
甲乙日生男孩。
行人
信息先至，人后至。
诉讼
提出证据，对方会答应和解。

**又曰**：
事遂勿忧煎，春风喜自然。
更垂三尺钓，得意获鳞鲜。

**释义**：
人情间的冷暖，锦上添花的很多，而雪中送炭的很少，爲什么呢？跟红顶白是人之常情，而打落水狗也是人的劣根性，是根本上很难杜绝的。而以此签来说，事情得到如此的得意，因此不必再担心，如果能耐心一点守候的话，收益便必会更多。

# 第88签

**第88签：古人伯牙访友**

**签文**：
```
悔亡悔亡，空耗两难。
人在东，西何可遇，
莫劳心力待时遇。
```

**详解**：

**签文**：
悔亡悔亡，空耗两难。
人在东，西何可遇，
莫劳心力待时遇。

**解曰**：
春秋时代的伯牙善弹七弦琴，唯曲高和寡知已不多。后在江上鼓琴，得遇锺子期从琴音中知其喜乐，知其所指，故结爲兄弟。两人约期于某日叙会，不料锺已夭丧，伯牙访之不遇，在其墓前再弹奏一曲「高山流水」以爲绝唱，后伯牙更碎琴以表再无知音之人。求得此签者，空怀抱负，终不得展，凡事不吉，需处处小心提防。

**详解**：
谋望
仍未遇恰当时机。
钱财
财富如水月镜花。
婚姻
男女皆难觅知音，初婚不利。
自身
修善可免忧愁。
家宅
家运平常。
开业
即使有计划，亦要搁置。
迁居
起止皆不吉。
出行
访友不遇兼惹惆怅。
疾病
因忧患而引起，心病还须心药医。
六甲
虽安産，但产后须注意。
行人
有事阻滞，不至。
诉讼
要平心静气争取和解爲宜。

**又曰**：
喜乐喜乐，暗中摸素，
水自镜花，空中楼阁。
啣石填海，抛珠弹雀，
视而不见，反成耽搁。

**释义**：
全签皆喻示应遇而不遇，应逢而不逢的情况，水中之月，镜中之花，可望而不可得，精卫鸟啣石欲填平海，空劳气力，以珍珠来打雀，徙然浪费，正如月沉于海底，人在梦中，要勉力而爲亦不可得，终是画饼充饥。

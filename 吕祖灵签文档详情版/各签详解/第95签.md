# 第95签

**第95签：古人孟尝君被害**

**签文**：
```
已入笼，何可脱；
已落空，难觅着。
遇鸡鼠，自知觉；
病果起，勿服药。
```

**详解**：

**签文**：
已入笼，何可脱；
已落空，难觅着。
遇鸡鼠，自知觉；
病果起，勿服药。

**解曰**：
孟尝君爲战国时期齐国的相国，拥有食客三千。未失势时，得食客冯煖爲其在食邑薜市义，尽焚该地百姓契约债券，得万民感恩拥戴。后孟尝君被诬告逃亡，抵关卡时因未天晓开关，由门下食客扮鸡啼顺利出关，得保性命，后得冯煖之计再重掌相位。求得此签者，若广积阴德当能转祸爲祥，逢凶化吉，险中有救。

**详解**：
谋望
可成，大小依据于以前的部署。
钱财
多行善事而耗损。
婚姻
未至，犹待时机。
自身
广积善当可保身。
家宅
中段反复，后渐趋好。
开业
初期恐有亏损。
迁居
适宜，唯需他助力。
出行
有阻力，后顺利。
疾病
立刻延医无碍。
六甲
胎儿无恙。
行人
未至，在途中。
诉讼
双方和解爲宜。

**又曰**：
愁洛阳锦绣万花丛，烂漫枝头不耐风；
三五月明时易过，夕阳西下水东流。

**释义**：
一到春季时万花盛开，花团锦族，在枝头上花朵开放得十分烂漫，令人感到十分写意。但须防盛极必衰，在最灿烂时又会趋于平静，在这种情况下，必须处处小心，步步爲营，不能轻率大意，同时一定要居安思危，爲将来作出部署，才能转厄爲祥。

# 第12签

**第12签：古人刘阮遇仙**

**签文**：
```
十日坐，一日行；
矶头有水，不碍利前程。
问到如何境，刘阮天台不误人。
```

**详解**：

**签文**：
十日坐，一日行；
矶头有水，不碍利前程。
问到如何境，刘阮天台不误人。

**解曰**：
刘阮看破世情，慕道求仙，后游天台遇仙得成正果。求得此签者，对于自己追求的目标或理想必须坚定信念，一步一步地向前行，前途虽有幻景如海市蜃楼也不要爲其痴迷，最后自不然能历尽崎岖而到达自己的理想。如占者能放弃故土风情，往外地发展则会大吉大利，必有所成。

**详解**：
谋望
如有朋友合作则胜算更大。
钱财
现仍未到时机。
婚姻
提防中途会有变化。
自身
渐渐交运而兴发。
家宅
会有贵人扶持。
开业
与熟悉此道的人合作，即吉。
迁居
往海外地区更佳。
出行
远行吉，并有贵人扶助。
疾病
延医即可治癒。
六甲
是略爲迟些，但无碍。
行人
对方落实后必至。
诉讼
听从尊长的意见必胜。

**又曰**：
此去万里程，却遇见知音；
同心和共济，建立此奇勳。

**释义**：
有远大的前程，本却离乡别井，独自前往，然后奋斗一番事业。可幸在途中遇到朋友，大家志同道合，一路上同心共济，由于双方的互相激励，终于各自创出大的事业。故占者若能放弃目前名利，远赴外地图谋，则大吉大利，事事皆顺，可扬名于异域。

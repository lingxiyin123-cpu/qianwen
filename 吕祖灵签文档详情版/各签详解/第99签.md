# 第99签

**第99签：古人邓艾锺会入川**

**签文**：
```
可以转移，可以了结；
只恨运命屯邅，故此一得一失。
```

**详解**：

**签文**：
可以转移，可以了结；
只恨运命屯邅，故此一得一失。

**解曰**：
邓艾爲三国时人，爲魏之镇西将军，与蜀将姜维相拒，后同锺会分军灭蜀，因被诬告谋反被杀。锺会亦爲三国时魏国人，官至司徒，爲司马昭重要谋士，公元二六三年，与邓艾分军灭蜀。蜀将姜维假降于他，两人密谋反魏，次年事败被杀。求得此签者，凡事不要急躁，自然有益无损，即使失去先机，也不可萌生退念，小心地进取，尚有可爲之事。因此纵有灾晦祸患，也可无碍，是先凶后吉的徵兆，由于命运之顺逆，而生出一得一失之势。

**详解**：
谋望
循步渐进，先难后易。
钱财
急则不成。
婚姻
慎防所托非人。
自身
修心即无大碍。
家宅
家运平常。
开业
须守成一段时间，急则立败。
迁居
无妨。
出行
顺利兼有小成。
疾病
初痊愈，后急转直下。
六甲
慎防流産。
行人
即至。
诉讼
已方有利。

**又曰**：
虎在路途边，行人莫乱呼；
路旁须仔细，灾祸自然无。

**释义**：
老虎横在路途之上打瞌睡，对人本是极爲凶险之事，因它一醒觉便会噬人。但只要行人小心谨慎，不惊呼狂叫，把睡着的老虎弄醒。这样，你慢慢的从它旁边经过，也是有惊无险，不会发生灾祸的，故平心息气，爲此签的重大啓示。

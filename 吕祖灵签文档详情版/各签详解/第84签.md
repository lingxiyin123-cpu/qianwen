# 第84签

**第84签：古人蒙正赏月**

**签文**：
```
秋风正好月当中，忽被雨朦胧；
如何往，难成功；不宜谋，须神工。
```

**详解**：

**签文**：
秋风正好月当中，忽被雨朦胧；
如何往，难成功；不宜谋，须神工。

**解曰**：
蒙正未发迹时，因苦至极，三餐不继。忽见好月当头，虽言赏月，实质触景愁思，满怀浩歎，何赏之有？正如「黄莲树下弹琴」——苦中作乐也！求得此签者，名利未遂，如月上中天，忽然下了一阵雨，弄得月夜迷朦，际此困顿之时，一切宜养晦爲妙。运势会在进入冬季时渐趋好转。

**详解**：
谋望
机会颇多，但爲假像。
钱财
入冬以后转好。
婚姻
最初似乎很好，实际不是。
自身
养晦修身爲妙。
家宅
尚须艰辛渡日。
开业
宜守业，创业不宜。
迁居
以不搬迁爲吉。
出行
访友不遇，谋事不成。
疾病
良好，进展中忽然转坏。
六甲
顺産。
行人
羁留原地未动。
诉讼
会吃亏，宜息纷争。

**又曰**：
一事已成空，一事还成喜；
若遇草头人，祸起萧牆里。

**释义**：
此签运势有若梅花间竹式，一事成空，一事成喜。空与喜交迭而生，令人徒费力气。在这段时间更要小心草头姓的人，所谓草头人，即姓黄、董、蒋、范、蔡等，若与他们在一起，很容易无端起祸，危及自身的安全，故一切以守爲宜。

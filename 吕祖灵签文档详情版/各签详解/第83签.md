# 第83签

**第83签：古人张士贵投军**

**签文**：
```
马已疲，人又疾；
天不从人，何须亟亟。
```

**详解**：

**签文**：
马已疲，人又疾；
天不从人，何须亟亟。

**解曰**：
张士贵贫无立锥，于是投笔从戎，初爲马步先行，累积颇多之功勳，但均爲主帅所冒夺去。其中一次单骑救主，亦只不过是多一点赏赐而已！但至最后，亦终能扬名吐气，爲国家立下功劳，爲自己奠下名利。求得此签者，初时命运不济，后则春风得意，所以初段时间，诸事不吉，故不要强求名利，假以时日，终必有所作爲。

**详解**：
谋望
转业或有生机。
钱财
周转颇爲艰辛。
婚姻
已婚者恐有分离。
自身
暂捺抑自己欲望。
家宅
宅运未得安稳。
开业
不宜，未能如愿。
迁居
以不变动爲宜。
出行
延期爲宜。
疾病
注意潜伏着的病患。
六甲
未有怀孕徵兆。
行人
现在不会来，要迟些。
诉讼
谈判恐生意外之事情。

**又曰**：
命蹇兮时违，灾及兮身羸；
皇天今不佑，嗟我兮何爲。

**释义**：
一个人的命运在蹇滞之时，其所得的机缘也是相违背的，当灾祸降临在他身上时，可能当时他的身体情况也是很羸弱的。爲甚么呢？这就是「祸不单行」的原故，亦即是因果的循环。既苍天都不予以庇佑，予以关顾，自己就不亟亟地营谋，首先打好自己的基础，充实好己，以候时机一到，自然水到渠成。

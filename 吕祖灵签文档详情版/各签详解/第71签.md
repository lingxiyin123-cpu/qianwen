# 第71签

**第71签：古人黄巢归天**

**签文**：
```
黑云迷径人难走，桥断风吹舟又危；
莫思徘徊，且去自思维。
```

**详解**：

**签文**：
黑云迷径人难走，桥断风吹舟又危；
莫思徘徊，且去自思维。

**解曰**：
唐末时盗贼黄巢作乱，率衆百万由湘水北上，攻克洛阳，后进入长安即皇帝位，国号爲大齐，改元金统。至金统五年，战事失败，退至泰山狼虎谷被围困，自杀死。求得此签者，乃笼中困鸟之象，凡事不吉。如能修德行善，见义勇爲，则可免灾危之厄。

**详解**：
谋望
暂不宜进取。
钱财
防资金周转不灵。
婚姻
强配的婚姻，迟些会离异。
自身
修德行仁可免灾。
家宅
防家中有灾厄。
开业
一切新的计划，中止爲宜。
迁居
迟则较宜。
出行
远行不利。
疾病
发病急者要注意。
六甲
顺産，但产后要小心。
行人
人不至，言信亦沓。
诉讼
谈判费时而且失败。

**又曰**：
东廓当年乞餕馀，一朝温饱丧其初；
人心不足蛇吞象，天理难忘獭祭鱼。

**释义**：
一个人立身处世必须要知足，不要贪图欲望。把自己的思想和行爲都深入欲海之中，这样会使人的性格及价值观完全摧毁。如东廓先生当年贫无立锥，以乞维持生活，一朝得志即忘记其初段时间的艰苦，使自己受欲望的枷锁牵着来走，终至不能自拨。因此只要不自暴自弃，全心全意地干，也不要忘本，最后都可以得到成功。

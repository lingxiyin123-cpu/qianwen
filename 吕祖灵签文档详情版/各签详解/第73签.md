# 第73签

**第73签：古人王道人收妖**

**签文**：
```
鬼迷神不佑，须求天神救；
立善行功成，好事方成就；
三接其画，多许诚求。
```

**详解**：

**签文**：
鬼迷神不佑，须求天神救；
立善行功成，好事方成就；
三接其画，多许诚求。

**解曰**：
唐朝李小二爲人鄙吝，损人利已。好探听人之险私，然后广爲宣传，爲乡里所不齿。一日在荒郊爲妖所惑，返家后即迷惘浑噩，终日痴痴呆呆，后其妻闻得王道人功法精深，遂前往恳请王道人出山收妖，王道人感其妻意诚，遂替李小二除妖，求得此签者，宜改过行善，回头是岸，则可获天福。

**详解**：
谋望
有贵人协助则可。
钱财
有妨碍，不能急急得着。
婚姻
慎防决定时会起风波。
自身
修身可消恶业。
家宅
慎言可享安乐。
开业
现在仍未是时候。
迁居
最终会不吉。
出行
中止外出爲宜。
疾病
提防久病会难痊愈。
六甲
安産。
行人
不能好来。
诉讼
自己一方会败诉。

**又曰**：
塞翁失马不爲忧，得马翻招长者愁；
祸福转圜无一定，须从倚伏究根由。

**释义**：
祸福虽然说是由天所界定，但细心想一想，绝对是由人来取决的，这就是有其因必有果的缘故。而且亦是古语说：「祸兮福所倚，福兮祸所倚」的结果。因此塞翁失马不必以此爲忧，得马亦不必以此爲喜，因爲其中所包含是福是祸仍未清晰。但只要时常修身，反省自己则可保一切平安。

# 第63签

**第63签：古人白帝城托孤**

**签文**：
```
龙已飞，虎已归，笙歌已歇；
兴而不久，万事如灰。
```

**详解**：

**签文**：
龙已飞，虎已归，笙歌已歇；
兴而不久，万事如灰。

**解曰**：
三国时蜀汉主刘备出兵伐吴，被东吴大将陆逊火烧七百营，忧病不起。诸葛亮至白帝城往见刘备，刘备将其子阿斗（即刘禅）託付予诸葛亮，并拜亮爲相父，以辅政事。求得此签者，凡事皆空之象，运乖才高者亦无高升之象，唯修身修德之人或可转危爲安。

**详解**：
谋望
可成，但须迟延。
钱财
托人去办，可以成功。
婚姻
左托右托，最终也会失败。
自身
修德或可转危爲安。
家宅
慎防病厄之劫。
开业
合五行者吉。
迁居
选吉方才可。
出行
 远近都不利。
疾病
很难治癒。
六甲
难産，小心调理。
行人
即至。
诉讼
托人去办和解可成。

**又曰**：
群凶弥六合，日月蔽烟尘；
修德从今始，犹难免祸侵。

**释义**：
当命蹇时迟，一切不如意事都会接踵而来，灾殃病患，也会因以前积下的因而生出今时的果。以前行善积德者当然是遇难有救，但如果从前未种下如此善福的人，就颇爲凶了，自今日起修善修德，虽然是「临急抱佛脚」，但亦可获得一定的庇佑。

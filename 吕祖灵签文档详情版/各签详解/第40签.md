# 第40签

**第40签：古人桃园结义**

**签文**：
```
富贵功名各有期，龙虎相逢鸡犬宜；
果煞采获仙人手，自是金麟上玉时。
```

**详解**：

**签文**：
富贵功名各有期，龙虎相逢鸡犬宜；
果煞采获仙人手，自是金麟上玉时。

**解曰**：
三国时代，刘备、关羽、张飞三人志同道合，希望能干出一番大事业，遂于桃园结义爲异性兄弟。后果然凭着三人的努力，建立蜀国，与魏、吴成鼎足三立之势。求得此签者，凡事大吉，名成利就，尤其是合伙者其利益更爲丰厚。

**详解**：
谋望
有人辅助则可成。
钱财
必可如愿以偿。
婚姻
快则有利。
自身
心存正直即佳。
家宅
和洽谈共处兴家。
开业
与人合伙大吉。
迁居
远近都吉。
出行
虽有阻，但终获利。
疾病
延医即痊愈。
六甲
可生男孩。
行人
寅辰日可至。
诉讼
在谈笑中可以解决。

**又曰**：
两人各合见交情，二人同心利断金。
鲍管陈雷今已往，羡君终解古人心。

**释义**：
古人说：二人同心，其利断金，几个志同道合的人合在一些，其团结出来的力量是相当大的，办起事来特别容易成功，如管仲、鲍叔牙二人爲好朋友，同时以共同做事，所以在辅助晋文公时使得晋文公成爲霸主，这就是和合见交情的含义。

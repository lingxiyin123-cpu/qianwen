# 第55签

**第55签：古人王守仁归隐**

**签文**：
```
月在天心人在船，名已将得，
利已将偿，何不耐烦，
不久起床，也要斟详。
```

**详解**：

**签文**：
月在天心人在船，名已将得，
利已将偿，何不耐烦，
不久起床，也要斟详。

**解曰**：
明朝至守仁，早年因反对宦官刘瑾，被贬爲驿丞，后归隐在故乡筑室于阳明洞中，故世称阳明先生，提倡「知行合一」。后平定「宸濠之乱」，官至兵部尚书。求得此签者，凡事见机行事则大吉，即应进则进，应退则退，谨慎即无灾至。

**详解**：
谋望
可成，但须待时机。
钱财
金钱调动无困难。
婚姻
仍未是时机。
自身
修心修身可得利。
家宅
安全。
开业
妄自创业，必至失败。
迁居
以中上爲佳。
出行
在途中会意外破财。
疾病
延医即可痊愈。
六甲
安産，多生女孩。
行人
虽迟也会到来。
诉讼
胜算不多，以和爲贵。

**又曰**：
富向勤中得，贵从修里来；
人能躬此理，灾去福星来。

**释义**：
富与贵，是每个人都希望得到的，究竟怎样可以取得富贵呢？道理很简单，但要做到则绝不简单。勤力工作能致富，修心修身得贵来。除此以外，始能戒除贪念，待人和气，谨慎行事都可以协助你取得富贵，所以能有恒地坚守上述条目，灾殃必去，福星常来。

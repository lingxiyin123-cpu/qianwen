# 第52签

**第52签：古人仁贵父子相会**

**签文**：
```
虎坐中堂，鸡太相闹；
天不如人求，反增多少笑。
安而不危，菊花天到。
```

**详解**：

**签文**：
虎坐中堂，鸡太相闹；
天不如人求，反增多少笑。
安而不危，菊花天到。

**解曰**：
薛仁贵爲求名利，遂投军从戎，经过连年征战，终于凯旋而归，衣锦还乡，遇见子丁山射雁，仁贵见虎而误射中丁山，后爲神人所救。求得些签者，做事切勿急躁，以防内忧外患交并而至。犹幸有神明协助，虽有险而无妨，凡事须当谨慎，好运秋天才到。

**详解**：
谋望
可成但很迟。
钱财
有障碍，不顺利。
婚姻
恐有不能成就。
自身
切勿急躁行事。
家宅
家中会有争拗。
开业
非本行终归失败。
迁居
现居吉，迁移凶。
出行
近处无大碍。
疾病
因疏忽而成大病。
六甲
安産，可生女孩。
行人
已在途中。
诉讼
托第三者和解爲宜。

**又曰**：
天道本无私，良心各自持；
倘然全昧却，后况更难知。

**释义**：
天道即人道，故人凭着自己的良心来做事是有神明鉴听的，这就即是世说的因果，所以天道是无私的。只对人的反应是基于人的所作所爲而定，倘然良知完全泯灭，那么往后的结果会是怎样呢？须记着，祸由自招，求天福不如求自己了。

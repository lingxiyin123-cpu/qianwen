# 第34签

**第34签：古人龙马负图**

**签文**：
```
自从河洛周天数，演到出门交有功；
一生际遇须龙马，不用踌躇自合行。
```

**详解**：

**签文**：
自从河洛周天数，演到出门交有功；
一生际遇须龙马，不用踌躇自合行。

**解曰**：
伏羲时代，有龙马出自黄河，背负一图，是爲河图。河图之数，天一生水，地六成之，居于北，爲北方壬癸水，阳一与阴六相配，象徵阳气始生。因此河图可说是敬受天命而来。求得此签者，正处运至时来，宜发愤努力向前，以达前途无量之境。

**详解**：
谋望
发愤向前当有所成。
钱财
会有七八分把握。
婚姻
现仍至最佳时期。
自身
努力前进而有贵人。
家宅
家宅运势渐至昌隆。
开业
无阻滞，但须努力，后会成功。
迁居
与家人商讨后无妨碍。
出行
远近皆且，有贵人助力。
疾病
急病宜防，旧病复发则无妨。
六甲
多数生女孩。
行人
已在时途，即至。
诉讼
小心处理，有败诉之象。

**又曰**：
一二三四五六七，八九相乘数乃毕；
老阳未变不能全，占者逢之静则吉。

**释义**：
一二三四五爲生数，六七八九十爲成数。生数不可移，成数可以移，两者之间有着相生相成的关係。一奇、一偶两组数的配搭，表示世间万物皆由阴阳化合而生成。乾爲老阳，老阳未变则不能乾坤生合，占之者遇此，万事宜静不宜动，静则吉。

# 第24签

**第24签：古人陈万年再世姻缘**

**签文**：
```
说到龙蛇马又来，看看无不遂心怀；
前生俱是有缘份，及到今时运已该。
```

**详解**：

**签文**：
说到龙蛇马又来，看看无不遂心怀；
前生俱是有缘份，及到今时运已该。

**解曰**：
陈万年家境贫寒，家徒四壁，其妻差点饥饿而死，后发愤求学，经十年寒窗，终得高中进士，并与其妻劫后重逢。求得此签者，先凶后吉，在辰巳午年月好运至，营谋无不遂意。但需要谨记，占得此签的人要坚守心志，方可得愿，若三心两意，纵有富贵亦不能久享。

**详解**：
谋望
渐进可名利双收。
钱财
稍迟可得富贵。
婚姻
完全没有阻碍。
自身
不争躁而爲则吉。
家宅
多点娱乐会较融洽。
开业
现在尚非其时。
迁居
以不动爲佳。
出行
暂时以不外出爲安全。
疾病
恐会有病缠身。
六甲
注意产后调理。
行人
在远方仍未动身。
诉讼
己方吃一点亏即能各解。

**又曰**：
几歴邪缘竟不摇，知君高见直冲霄。
世间共识阴功好，冥冥先把姓名标。

**释义**：
古人常说：「富易交，贵易妻」。当人取得富贵之后，很多人都会抛弃贫贱时的朋友与妻子。能够象张节一样，「贫贱之交不可忘，糟糠之妻不下堂」又有几人？所以在得意时，能够不动摇自己的心志，才是顶天立地的人，运气才能保持。

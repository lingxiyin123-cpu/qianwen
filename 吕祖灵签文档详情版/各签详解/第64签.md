# 第64签

**第64签：古人汉玉路逢拐骗**

**签文**：
```
生之时多乐，半路上下不相逢；
只看无妄之卦，方可悔吝分明。
```

**详解**：

**签文**：
生之时多乐，半路上下不相逢；
只看无妄之卦，方可悔吝分明。

**解曰**：
汉玉长于富贵之家，未知世逢凶险，携金外游，途中被人尽骗金钱。求得此签者，自身作事须小心谨慎，提防外出途中有惊险之事发生，弄至恐惧、忧虑尽现眼前。幸而善人占得之无碍，存心正直者占之亦吉，完全视乎个人的德行而有所不同。

**详解**：
谋望
阻碍重重。
钱财
恐防财耗。
婚姻
托第三者以求会加速。
自身
修善可消劫。
家宅
有无妄之灾。
开业
不宜，防有意外。
迁居
不妨。
出行
远近都不利。
疾病
小心会变重。
六甲
产后需注意调理。
行人
将至，有同伴。
诉讼
双方以和爲贵，能解决最好。

**又曰**：
恃富因忘本，贪婪竟失中；
欺心天不佑，神道岂私凶。

**释义**：
富贵本爲镜中月，水中花，往往很多人「三更穷，五更富」，对于很多本来爲贫苦的人家，一旦发迹起来，很容易忘记当日自己捱穷的日子，对于贫苦的人其怜悯心也不会很大，所谓爲富不仁，因此这些人很多时会爲维持自己的财富而做欺心之事，贫富其实只在于知不知足，能不欺心则天必得佑，祸必不侵。

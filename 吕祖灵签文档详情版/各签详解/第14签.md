# 第14签

**第14签：古人牛皋战兀术**

**签文**：
```
一虎出山，一龙飞天。
天缘人缘，上国收贤。
生育之美，造化两全。
```

**详解**：

**签文**：
一虎出山，一龙飞天。
天缘人缘，上国收贤。
生育之美，造化两全。

**解曰**：
宋朝勇将牛皋，力拒金将金兀术大军之进犯，遂大战兀术。龙虎相斗，一定有所损伤，究竟是那一方会受到严重的灾害呢？就要看双方的造化谁属。求得此签者，可以乘时而起，既易且速，但切勿轻易与人争吵，以免转盈爲亏。

**详解**：
谋望
如遇时机可得成就。
钱财
有第三者帮助会成。
婚姻
一定可以成就。
自身
顺应时机以修身。
家宅
慎防有口角之争。
开业
没有什么障碍。
迁居
移居无妨。
出行
远近皆宜。
疾病
如果拖延就很危险。
六甲
略爲有障碍，生女孩。
行人
信息来，但人未来。
诉讼
双方清楚利害得失，会和平解决。

**又曰**：
鹏鄂当秋势转雄，乘风分翼到蟾宫。
荣华若问将来事，先后名声达九重。

**释义**：
鹏鸟与鄂鸟都是一飞万里的神鸟，当其展翅腾飞之时，虽然远到月宫都是可以的。最重要的问题是能够乘时而时，这样就会很容易和很迅速了。这是「时势造就出的英雄」，所以在碰到这种时机时，他日的成就，亦是无可限量的。

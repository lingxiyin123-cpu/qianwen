# 第94签

**第94签：古人蒙正捞斋**

**签文**：
```
事已如此，悔之何及；
纵有不祥，还宜阴骘。
```

**详解**：

**签文**：
事已如此，悔之何及；
纵有不祥，还宜阴骘。

**解曰**：
蒙下家贫，但苦学不断，爲求得到饱暖，专心致志，于是寄住寺院，以求两餐。依寺院规定，饭前敲锺，由于寺内有些和尚轻视蒙正清贫，于是有一次在饭后才敲锺，使他无食。蒙正自此更艰苦读书，后得中状元。求得此签者，能修德行仁，获福当在其后，此乃先苦后甜，先难后易之象，应多积阴骘才可。

**详解**：
谋望
中途会有变化。
钱财
仍未可以获得。
婚姻
还是不成。
自身
修品德，行仁义当可获福。
家宅
家运暂时不佳。
开业
时间仍未适宜。
迁居
会因种种原因而搁置。
出行
须小心提防口舌之争。
疾病
慢性病，长久可痊愈。
六甲
吉利。
行人
还在原地未啓程。
诉讼
对己方不利。

**又曰**：
愁面笑顔开，忧心事可谐；
但凭依理做，不必费疑猜。

**释义**：
每一件事情，到达了某一个地步，就应依着其理法来进行，不必再想或后悔当日爲何不如此如此的去做。纵使是绕路而行，终必有成。若一味转变行事方针，则只不过是原地踏步，再无进展之势。故此依理而做，而不必再有猜疑。

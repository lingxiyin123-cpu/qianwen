# 第23签

**第23签：古人煮酒论英雄**

**签文**：
```
海上一龙飞，春风得意回；
从此欣荣遇，只待一壬来。
```

**详解**：

**签文**：
海上一龙飞，春风得意回；
从此欣荣遇，只待一壬来。

**解曰**：
曹操刘备共宴，当时曹爲汉之大臣，挟天子以令诸侯，刘仍没有一兵半卒。边饮边说时，曹与刘谈论政治人物谁爲英雄。当时谈及刘及孙权，刘听罢，汗流浃背，恐曹加害。求得此签者，虽危中有救，亦恐有无妄之灾，凡事三缄其口爲妙。

**详解**：
谋望
现仍观望，稍后有进取之机，但要略施计谋，隐藏实力，才能在尔虞我诈之局中取胜。
钱财
很费力才能获益。
婚姻
暂未至时机。
自身
以慎言修身。
家宅
防生无妄之灾。
开业
现在不是时候，中止计划爲好。
迁居
无妨，但迟此更好。
出行
在目的地会吃点苦。
疾病
延医即可治癒。
六甲
略爲有阻碍。
行人
已在途中。
诉讼
双方会迅速和解。

**又曰**：
芸窗虽说有深功，无奈朱衣不肯从；
丹桂杏花相逼近，羡君此日定峥嵘。

**释义**：
吃得苦中苦方爲人上人，因十年寒窗苦读仍未折桂，始终都不能说是有所成就。犹如人努力在发展自己的事业，但未有寸进，仍然是云遮月头，但如果能得到突破，则一雷天下响，吐气扬眉了，必然惹来豔羡目光。占此卦得运之时应在夏末初秋。

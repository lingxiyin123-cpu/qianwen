# 第1签

**第1签：古人王母祝寿**

**签文**：
```
蓬莱东阁玉桃香，顺水行舟仙赐方；
宜男正好图全计，不必他方卜地长。
```

**详解**：

**签文**：
蓬莱东阁玉桃香，顺水行舟仙赐方；
宜男正好图全计，不必他方卜地长。

**解曰**：
王母娘娘大寿，各仙家前往致贺，场面极爲浩大。王母对贺客皆赐以仙桃仙丹等长生不老的美食。衆仙大会，乐韵钧天，属吉庆徵兆。

**详解**：
谋望
不要急于成事，须待时机。
钱财
委託异性去处理，收穫较易较佳。
婚姻
虽吉，但仍要守候。
自身
修善即可平安。
家宅
家宅平安，逐渐昌盛。
开业
与朋友合资易有所成。
迁居
选吉方即可。
出行
往远方，有意外之喜。
疾病
一切小病痛都能痊愈。
六甲
顺产，生男孩。
行人
音信将至。
诉讼
双方可得和解。

**又曰**：
占得鳌头百事成，逢迎到处不须疑；
从兹修省能方便，福禄绵绵自可期。

**释义**：
在运势尚未通达的时候，当然是有志难伸，一旦鳌头独佔。跟红顶白、锦上添花的人会接踵而来，如果一下子利欲薰心，很容易把大好前程新自断送。这段时候最 好是自我签讨，保持稳健踏实的态度，才能够使到福禄绵绵不绝。

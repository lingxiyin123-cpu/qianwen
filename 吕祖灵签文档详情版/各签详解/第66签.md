# 第66签

**第66签：古人姜子牙卖卦**

**签文**：
```
得失皆天定，忧喜总归命；
知天知命，守之爲幸。
```

**详解**：

**签文**：
得失皆天定，忧喜总归命；
知天知命，守之爲幸。

**解曰**：
以笃信天命来说，一个人的得与失皆由天所定。即如相助周文王灭商的姜子牙，在未得文王器重时，所遇之生活亦是非常艰苦，是靠着人家占卦算命爲生。求得此签者，万事宜慎，切勿轻躁行事，如能耐守待时则吉。

**详解**：
谋望
谨慎待时则可。
钱财
暂未有财运。
婚姻
不作无理争论可成。
自身
以安分守己持身。
家宅
宅运平澹而已。
开业
不要强求去做，防有凶险。
迁居
吉方便无妨。
出行
有阻碍。
疾病
宜急延医治理。
六甲
生男孩。
行人
未有动身的迹象。
诉讼
宜息争，由有力人十协助则速。

**又曰**：
积德由来胜积金，羡君世代有簪缨；
残年底事行阴险，雪虐风饕末路人。

**释义**：
有积蓄是人人所喜爱的，有钱的人积金，有德行的人则积德。积金是有形的、积德是无形的，但古人言，功名利禄皆由积德之士所有，故积德亦可以成爲有形的回报，只要能安份纪，处处以守爲重，则不会迷失心性。

# 第46签

**第46签：古人蔡中兴修建洛阳桥**

**签文**：
```
桥已断，路不通。
若要通时，也候三五之中；
许下善功息不穷。
```

**详解**：

**签文**：
桥已断，路不通。
若要通时，也候三五之中；
许下善功息不穷。

**解曰**：
蔡中兴乐善好施，常作修桥筑路之事。当时洛阳桥破烂，行人过之诸多不便，故发宏愿筹款修建此桥。由于精诚所至，观音化银相助，终必修建洛阳桥。求得此签者，做事只要诚心则可感动天地，终必有意外之喜，收成必佳。注意子、亥年月日，会是吉昌之时。

**详解**：
谋望
先艰后易。
钱财
有人的帮助得财。
婚姻
仍未有成就之意。
自身
多行善事以培福根。
家宅
凡事神心则吉。
开业
必能成功，后来大佳。
迁居
吉，但须先装饰。
出行
旅行则无妨。
疾病
延医治理可愈。
六甲
会有阻碍。
行人
在远处来会较迟。
诉讼
说明利害得失，可以和解。

**又曰**：
身离渡口箭离弦，人在前途眼望穿；
莫怨行人归棹晚，管教日内得团圆。

**释义**：
身离渡口即在江中，箭离弦即已发箭，喻示事情已在进行之中，还可以走回头路吗？当然是不可以了，所以只有鼓其馀勇，继续向自己的目标进发。同时千万不要怨歎做事的收成太迟，上天一定会清楚你的行善之心而给你一个满意的答覆。

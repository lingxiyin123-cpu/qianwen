# 第97签

**第97签：古人范仲淹卖字**

**签文**：
```
吉凶相半人宜识，莫任劳劳损无益；
有时天运一回还，自然天道无痕迹。
```

**详解**：

**签文**：
吉凶相半人宜识，莫任劳劳损无益；
有时天运一回还，自然天道无痕迹。

**解曰**：
范仲淹爲北宋时人，擅长于诗词散文，文章有丰富的政治内容。少时贫困，但力学而苦读，两餐无继，于是上街卖字维生，至大宋祥符时考得进士，自此官到宰相，位极人臣。因此，凭你才高八斗，满腹经论，只要时运不济，亦不能有所作爲，一旦天运回还，荣宠立见，了无退缩。求得此签者，初期运虽蹇，后必得亨通。

**详解**：
谋望
仍须等侍时机。
钱财
恐周转不灵。
婚姻
姻缘虽好，但未至。
自身
谨慎可无碍。
家宅
修善可得平安。
开业
初段艰难重重。
迁居
移居吉。
出行
会停滞数天。
疾病
延医即可痊愈。
六甲
生男孩。
行人
渐至。
诉讼
再候一段时间谈判，可无碍。

**又曰**：
知君久欲觅良缘，自有嫦娥下九天；
但使河东安睡稳，莫教风急散文鸳。

**释义**：
心中已萌生了营谋之念头，不必急躁也不必担心，希望是可以得到实现的，而且成绩也是不错的。须注意，重点是在时间和成就上，因此只要不冒然求进，终归可遂心愿，若果急于求成，反爲不美。

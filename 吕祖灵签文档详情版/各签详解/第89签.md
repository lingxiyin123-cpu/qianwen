# 第89签

**第89签：古人七擒孟获**

**签文**：
```
隔了几重山与水，前途终是隔难行；
只待草头人，方可两成谋。
```

**详解**：

**签文**：
隔了几重山与水，前途终是隔难行；
只待草头人，方可两成谋。

**解曰**：
三国时代，诸葛孔明爲承继刘备的遗愿，北定中原，所以考虑伐魏，当时南方蛮人经常搔扰蜀中，孔明欲先平定南蛮，再图北伐。南峦主倾心信服，终其一世不再犯境，求得此签者，事虽成但要歴尽艰辛。而且若能与草姓人商量谋事，成功的机会会更大。

**详解**：
谋望
终可成，但艰辛劳碌至极。
钱财
将有损耗。
婚姻
防中介之人反复。
自身
苦尽甘来。
家宅
未得安宁。
开业
初期辛苦奔波。
迁居
须迁数次始吉。
出行
前途略有险阻。
疾病
初发即须延医。
六甲
可生女。
行人
没有来的意愿，在家守候着。
诉讼
经多番交涉始成功。

**又曰**：
船到江心浪拍天，羡君飞渡得安然；
紧收蓬脚牢拉舵，尚有前途十八滩。

**释义**：
船到江中心突然遭遇波涛胸涌，拍天而起，这种情况非常之危险，可幸此次情况是有惊无险，终必安然渡过。但虽然如此，仍然不能大意，要作好一切安全措施，使船稳定地向前行，因爲前面还有十八个险滩要渡过呢？故此签喻示作事有阻滞但无碍，但仍然要谨慎地贯彻始终，以避免功亏一篑。

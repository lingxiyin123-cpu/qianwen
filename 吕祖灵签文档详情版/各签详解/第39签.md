# 第39签

**第39签：古人三顾草芦**

**签文**：
```
鞭策长安路，天仙第一班。
已及时，春风桃李姓名奇。
一枝独佔梅花上，次第春风到草芦；
三十六宫春，上国定观光。
```

**详解**：

**签文**：
鞭策长安路，天仙第一班。
已及时，春风桃李姓名奇。
一枝独佔梅花上，次第春风到草芦；
三十六宫春，上国定观光。

**解曰**：
三国时候的诸葛孔明号称「卧龙」，未出草芦已三分天下。刘备慕其名三顾草芦，礼贤下土，恳请孔明出山相助，以争霸天下。孔明感其诚，遂出山；后造成魏、蜀、吴鼎足三立之局面。求得此签者，由于自己的才能受人尝识，因此可以给自己一展才能的机会，凡事大吉。

**详解**：
谋望
胸有成竹，待时而已。
钱财
要再三托人始成。
婚姻
双方误会，难成。
自身
修身修心爲上。
家宅
渐有昌隆之势。
开业
现在不是时候，多等一段时期便吉。
迁居
不搬动的好。
出行
初段不顺，后大吉。
疾病
须延专科医生治理。
六甲
多数是男孩，而且顺産。
行人
已在途中。
诉讼
能尽量和解最好。

**又曰**：
龙潜于渊，或跃于渊；
甘霖四野，遍泽大千；
煦妪覆育，万物含贝。

**释义**：
人未得时，如龙潜于深渊，蛰伏不动，得时则跃出于渊，大展所长。而其德才华，将会遍泽大千世界，万物受其覆育而欣欣向荣。故凡事要坚忍耐劳，以待时机。

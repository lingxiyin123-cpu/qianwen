# 第65签

**第65签：古人杏元和番**

**签文**：
```
一天星斗灿辉煌，风雨未光；
起而不扬，也要两三番。
```

**详解**：

**签文**：
一天星斗灿辉煌，风雨未光；
起而不扬，也要两三番。

**解曰**：
古代匈奴屡犯边境，爲免两国兵戎相见，祸及无辜的百姓，于是用和亲政策，送出杏元出嫁匈奴王，由于结有姻亲的关係，从此平息两国的纷争。求得此签者，虽处凶险亦藏有象，只要事事以谦和爲本，则诸事到头来都会无妨也。

**详解**：
谋望
忍耐守候则有成。
钱财
可以托人取得。
婚姻
平顺谐和。
自身
修善持德则佳。
家宅
宅运昌隆。
开业
与有经验者合作则吉。
迁居
吉，处处无妨。
出行
非常顺利。
疾病
初病无妨，久病恐不治。
六甲
略有妨碍，亦会安産。
行人
还有一事要阻延，迟些到。
诉讼
双方已有协议和解。

**又曰**：
久生尘埃心已灰，无端富贵又未催；
盐梅自是调羹用，那许凡夫坐井窥。

**释义**：
当人对每一件事都毫不牵挂地放下时，他实际已到了一个返璞归真的地步，一切名利、荣辱都已是身外之物，绝不会打动他们的心。但很多时你愈不追求的，它愈是追着你，又不是人所可控制，取捨如何？要视乎自己的情况。总之，即是事情看起来已经到了尽头，但最后都会有柳暗花明之象出现，一切无妨。

# 第54签

**第54签：古人刘备借荆州**

**签文**：
```
一半就，一半不就，惹得争斗。取之不合，治之宜早；吉祥方保，也要慢寻讨。
```

**详解**：

**签文**：
一半就，一半不就，惹得争斗。取之不合，治之宜早；吉祥方保，也要慢寻讨。

**解曰**：
三国时代，魏、蜀、吴未鼎足三立。刘备仍未有立足之地，后从军师诸葛孔明的建议，刘备取荆州，遂得以建基立业。求得此签者，须看自己爲何人物，如属刘备，则凡事大吉，因有创业之基地。如属借出荆州者，则此借出再无收回之象，属不吉。故爲一半可以，一半不可以之卦。

**详解**：
<B>谋望
一半成，一半不成。
钱财

**又曰**：
静则庶几，动则辄咎；狼跋其前，载疐其后。

**释义**：
做事皆有动静两方面，静爲思考策划，动爲执行。在处于目前的境况，静时未能取得法，动时又容易生出错误，人到此时，可谓狼狈至极，因爲有所阻碍而不能顺利前进。故宜以忍耐守成爲上，待时运一至，然后才顺势推行。

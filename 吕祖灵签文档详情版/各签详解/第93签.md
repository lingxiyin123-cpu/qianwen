# 第93签

**第93签：古人梁山伯路遥访友**

**签文**：
```
心急马行迟，妄干何有益；
且自安份，天神护吉。
```

**详解**：

**签文**：
心急马行迟，妄干何有益；
且自安份，天神护吉。

**解曰**：
梁山伯与女扮男装的祝英台同窗共读三年，感情非常好，但未知英台爲女性。英台返家前向山伯托言其妹之才貌与其一般，并许婚予山伯。唯山伯于约期后三天才至英台府邸，英台已被其父许婚予富豪马家。山伯忧忧而终，英台出嫁日往坟至祭，亦殉情而死。求得此签者虽安份守纪，有缘份当有机遇，此时应把握，倘时机一失，过后很难再次觅求。

**详解**：
谋望
会遭第三者介入而弄翻。
钱财
只得空想，无实际进展。
婚姻
难配难合。
自身
修心向善可免灾厄。
家宅
防有丧事入屋。
开业
仍未到时候。
迁居
不利。
出行
途中会有阻障。
疾病
如是急症恐有性命之忧。
六甲
防有産厄。
行人
突然音讯全无。
诉讼
败诉，恐有刑事之灾。

**又曰**：
六阴速遘象艰危，消息如斯莫强爲；
惟有齐心勤省疾，元关转处吉神随。

**释义**：
六阴若相逢，表示将发生灾厄凶危之事，此番透此的消息既然是这样，不必妄动，强要运转乾坤，凡事只要小心在意提防，勤加修饰己身，则自不然灾祸消弭，随着吉祥之神来运转。

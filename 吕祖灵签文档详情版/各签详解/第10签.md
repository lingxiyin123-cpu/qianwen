# 第10签

**第10签：古人古城相会**

**签文**：
```
问仙须要酬天地，百事如心定有利；
果能春秋两处好，龙虎呈祥，风云际会。
```

**详解**：

**签文**：
问仙须要酬天地，百事如心定有利；
果能春秋两处好，龙虎呈祥，风云际会。

**解曰**：
做每一样事情都要经过一些过程才能成功，所谓有因才有果。因此要遇仙的话。一定要先酬谢天地，以表明自己的真诚。求得此签者，初时必有异象，但如能处事心诚，勤俭剋己，则必风云际会而有所成。

**详解**：
谋望
勤俭克己则有成。
钱财
财富由天，有横财。
婚姻
双方合意，可以团圆。
自身
克勤克俭以修身。
家宅
渐渐趋向昌盛。
开业
能得贵人扶助。
迁居
选吉方便可。
出行
 无往而不利。
疾病
急病、慢性病皆可治癒。
六甲
顺産，可得男孩。
行人
已在途中。
诉讼
初是谈判困难，后得和解。

**又曰**：
云散月当头，牛前马后逢；
张弓方抵御，一箭定全功。

**释义**：
一个人不如意时，如月亮被云遮挡着而不能得见，一旦云散月落，皎皎的月色当然光照大地，令人感到开怀。对于很多繁琐的事情，会更付出心爲去处理，若一旦取得重点，而放手一搏，问题就会迎刃而解了。名利之收穫当在丑年月前，午年月后。

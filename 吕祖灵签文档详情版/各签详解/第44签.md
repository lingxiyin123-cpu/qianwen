# 第44签

**第44签：古人佳人隔帐纱**

**签文**：
```
莫孤疑，事在可疑。
有可疑，不疾不徐，
自有出头日子，当看金花一二枝。
```

**详解**：

**签文**：
莫孤疑，事在可疑。
有可疑，不疾不徐，
自有出头日子，当看金花一二枝。

**解曰**：
有夫妇二人，爲了逃兵乱而失散，闻得该地清山寺之神灵颇爲灵验，二人遂分别进入该寺。夫在寺内牆上诗：「早年不到清山寺，得到清山雨未休。」后妻见后，续句：「天地也知离别久，故将风雨爲君留。」求得此签者，宜心持善念，则可得贵人出面扶持；如怀恶则祸不远矣！

**详解**：
谋望
慎防被人反覆误事。
钱财
要守候才可成。
婚姻
有阻滞未成。
自身
修心向善则吉。
家宅
慎防口舌起祸。
开业
与人合伙不宜。
迁居
迟则有利，急不宜。
出行
远方适合。
疾病
迅速延医可治。
六甲
不必担心，是顺産。
行人
会到但迟。
诉讼
如能赔少许对方即可和解，此是福也。

**又曰**：
下蓍佔先机，迂迴路转迷；
目前只宜守，且免是和非。

**释义**：
在可以攻的时候攻，在应该守的时候守，如果能真正做到这一地步，就是知道进退了。以这一支签意的理解，现阶段绝对是只适宜守的，因爲一进就会産生很多是非口舌和争拗。爲保持自己目前尚不算差的运气，还是以守爲妙！

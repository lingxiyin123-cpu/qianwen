# 第61签

**第61签：古人平贵鸿雁传书**

**签文**：
```
鸿飞天外，舟动无风；
人有月明中，自有喜相逢。
```

**详解**：

**签文**：
鸿飞天外，舟动无风；
人有月明中，自有喜相逢。

**解曰**：
石平贵爲保卫国土，征西十八年，未得返家，靠鸿雁得书，以报平安。后完成征讨后回国，返家始得夫妻相会。求得此签者，有久屈而得申之现象，亦先凶后吉之意，使平来抑屈的命运，因突然接获佳音而得以舒展，正如舟楫无风自动一样。

**详解**：
谋望
甚么事情都可成功。
钱财
可得利益。
婚姻
双方和善。
自身
积善可发达。
家宅
宅运平常。
开业
初期无大进展。
迁居
任何一方都适合。
出行
有朋友同行吉。
疾病
很快痊愈。
六甲
顺産，生男孩。
行人
已回。
诉讼
己方可得胜诉。

**又曰**：
君问平生事业奇，前人积福后人宜；
东南西北皆如意，一路康庄任你驰。

**释义**：
前人种树后人凉，前人积福后人收，既然是已收前人的福荫，后人更应扮作前人，积福积善以留后代人享用。代代积福，代代可享，所谓积善之家，必有馀庆就是这个意思。运势亨通的，则不管何地何方，皆爲康庄之道，任你驰骋，因此修善积福至爲重要。

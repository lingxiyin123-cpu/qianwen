# 第7签

**第7签：古人盘古初开**

**签文**：
```
临物易与，天地人合；
一到春风，百祥骈集。
```

**详解**：

**签文**：
临物易与，天地人合；
一到春风，百祥骈集。

**解曰**：
天地溷沌，盘古初开，万物未齐，一切皆由天地慢慢地孕育出来。由于阴阳的涵盖，三才俱备，一到春风化育，万物祯祥。求得此签者，如能应时，则百事遂心，是大吉之兆，如春风未到，则困在这段时间，往后才能名利双收。

**详解**：
谋望
时机未到，半途恐有阻滞。
钱财
有障碍，难望急成。
婚姻
初婚不宜，再婚宜。
自身
修善平安可发达。
家宅
春风一至，百事吉祥。
开业
初段阻滞，慢慢转好。
迁居
延迟爲佳。
出行
小心选择时机。
疾病
治癒后不可不注意疗养。
六甲
宜十分当心。
行人
过几日有消息至。
诉讼
急则恐防败诉。

**又曰**：
随处周旋悉有神，善有善报恶逾嗔；
全凭栽得心田好，谁不虔求仗德仁。

**释义**：
「善有善报，恶有恶报」是人人都认识的语名，但如何能真正地体会到呢？就未必人人知道。简单来说，善就是心中常存有仁恕、行爲有德行，恶就是贪婪、嗔怒所引发。因此要有善的报应，就必须要种善的因，这种的话，即使遇上任何凶险都可以化解。

# 第17签

**第17签：古人张飞喝断长板桥**

**签文**：
```
一旦雷声震，四方皆有庆；
转吉又增祥，壬癸子午定。
```

**详解**：

**签文**：
一旦雷声震，四方皆有庆；
转吉又增祥，壬癸子午定。

**解曰**：
三国时关羽被曹操款待爲上宾，但不许其与刘备会面。后关羽得悉刘备的去处，遂潜走出关，过五关斩六将，至长板桥附近人马疲乏，幸得张飞接应。张飞力拒曹兵，一声吆喝，长板桥爲之折断，曹兵遂不敢再追。求得此签者，创业不会很艰难，但要守业则很辛苦，唯有以勤力、谦虚来处事即可。

**详解**：
谋望
能够坚守不怠惰可成，不可持一时之勇，最好得到有智谋的好友或拍挡扶持。
钱财
急则无结果。
婚姻
速战速决，不可一拖再拖。
自身
以勤、谦来处事。
家宅
积德可保平安。
开业
创业时颇爲顺行。
迁居
毫无妨碍。
出行
途中会有吉庆，但逗留日子稍长。
疾病
注意饮食卫生，可以痊愈。
六甲
顺産，会生女孩。
行人
子午日会到。
诉讼
儘量和解爲宜。

**又曰**：
鼠牙雀角本非奇，彼若欺人反自欺；
明镜高悬道理正，管教庭训得和宜。

**释义**：
有很多人常常用欺诈手段来取得人的信任或利益，以满足一已之欲。事实上，他所施展的手段，以爲很高明，其实往往弄巧反拙，欺人反自欺。只要能够在初期时有正确的宗旨来干，则往后就不会偏歪。

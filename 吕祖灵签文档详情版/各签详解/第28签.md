# 第28签

**第28签：古人赵子龙佐刘备**

**签文**：
```
前生原是此家人，今世相逢定有因；
龙虎呈祥遂天子，五三六二是奇期。
```

**详解**：

**签文**：
前生原是此家人，今世相逢定有因；
龙虎呈祥遂天子，五三六二是奇期。

**解曰**：
赵云字子龙，乃三国时常山真定人，初期辅佐公孙瓒，并忠心失志扶助汉室。曾以数十骑力拒曹操大军，有「一身是胆」的称号。求得此签者，功名富贵有神明护助，目前虽困，但终必逢时，如往东西方发展，营谋会得到丰收。

**详解**：
谋望
往东西方向发展可成。
钱财
周转绝无问题。
婚姻
婚后非常美满。
自身
勤勉修持可遂心意。
家宅
和气自能生财。
开业
勤苦可以有成。
迁居
怎样做也无阻碍。
出行
公干尤爲亨通。
疾病
虽久病也能痊愈。
六甲
顺産，可得男孩。
行人
即至，并有佳音。
诉讼
宜尽速解决，久延则恐有变。

**又曰**：
造化生来信自然，师征千里福绵绵；
功名得就神明助，蛇兔相逢定变迁。

**释义**：
人生的造化（即运气、福份）原有天定，但以自然平常之心来对待的话，自不然会感到事事都顺遂，而享得的福泽也会绵绵不绝。功名（官职）的成就也可以得到神明的帮助而取得。应验变迁的时间会在卯巳年月日。

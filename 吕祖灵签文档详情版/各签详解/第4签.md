# 第4签

**第4签：古人唐三藏取经**

**签文**：
```
正大有鬼神之助，吉祥成忠厚之报；
不怕邪魔小崇，只看秋收冬藏。
```

**详解**：

**签文**：
正大有鬼神之助，吉祥成忠厚之报；
不怕邪魔小崇，只看秋收冬藏。

**解曰**：
唐时唐三藏立志弘扬佛法，爲寻求正道，往西天竺取经，曆尽千辛万苦，涉足山川险阻，凭着坚持之心，鬼神之庇护，终于得偿所愿，满载而归。求得此签者，只要行事光明正大，忠厚待人，必有鬼神之助，逢凶化吉。求谋以秋季爲有利。

**详解**：
谋望
待到秋季时节有成。
钱财
春夏破财，秋冬有盈馀。
婚姻
暂时仍未到时机。
自身
坚心守志以修身。
家宅
行忠厚合家平安。
开业
选择秋冬季爲吉。
迁居
远方较合适。
出行
远处有贵人相助。
疾病
略有凶险，无碍。
六甲
生産顺利。
行人
已在途中。
诉讼
虽胜也要费很多时日。

**又曰**：
人心叵测却难防，幸有神明鑑在旁；
若使坚持心不改，一生偃蹇理应当。

**释义**：
人世间最难忖测的是人心，所谓「心猿意马」，一刹那间人的思维可以有很大的变化和波动，因此爲善爲恶也在一念之间。若能使自己时刻秉持忠厚、正大之心，则一生不会经历大凶险，亦不会有太大的蹇滞出现。

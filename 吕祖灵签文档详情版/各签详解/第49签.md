# 第49签

**第49签：古人陶渊明归隐**

**签文**：
```
牆头草，篱下花，
山穷水尽，未必繁华。
若要繁华，也要归家，
方可如心得奢。
```

**详解**：

**签文**：
牆头草，篱下花，
山穷水尽，未必繁华。
若要繁华，也要归家，
方可如心得奢。

**解曰**：
晋朝陶渊明生性恬澹，品格天然，不爲五斗米而折腰。由于爱好隐逸宁静的生活，放弃做官，归隐田园，过着飘逸的生活。求得此签者，若恋名利，则稍爲痛苦，因名利会渐渐退减，若心无挂虑，将名利看澹，则凡事先难后易，可得安享生活。但凡事皆以退让爲妙。

**详解**：
谋望
先观察清楚才决定。
钱财
手头金钱不很充裕。
婚姻
未到适当时候。
自身
以退让爲修身之本。
家宅
谦和一家平静。
开业
强求也不会成功。
迁居
远处较适宜。
出行
不是公干则吉。
疾病
须防病情转恶化。
六甲
须注意产后小孩健康。
行人
尚远而未至。
诉讼
以忍让来谈判，可解决。

**又曰**：
退稳高山自种田，结间茅屋自云边；
也知时运多颠倒，何必营谋惹祸缠。

**释义**：
世事如棋局局新，在人世间待人处事，无不爲蝇头小利而劳碌。只要环境许可，退隐泉林自耕自种亦是一个好办法。但这对大部份人来说是奢望。退而求次，做人处事以退隐的心情，处处以不争，以退让爲先，则可减少很多祸缠。

# 第38签

**第38签：古人鲤鱼化龙**

**签文**：
```
时行运遂不须愁，一到桃花皆自由；
果然渊内龙飞跃，脱俗超化始出头。
```

**详解**：

**签文**：
时行运遂不须愁，一到桃花皆自由；
果然渊内龙飞跃，脱俗超化始出头。

**解曰**：
在中国一处叫龙门的地方，鲤鱼全部逆流而上，只要跃过龙门这一关，就可以化成龙腾上天。故求得此签者，现时正处于交接运势的关头，一旦跃过龙门三尺浪，就会超凡脱俗，有大的作爲，正是否极而泰来，每事大吉。

**详解**：
谋望
急则不成，终必成。
钱财
托人可成。
婚姻
择定佳期爲佳。
自身
常行善事可积德。
家宅
家运正转趋好。
开业
暂坚守旧职吉。
迁居
选吉方便无碍。
出行
途中会遇贵。
疾病
因应病情而治疗，可愈。
六甲
顺産。
行人
仍未至，再催促即至。
诉讼
现在谈判不利，迟一阵吉。

**又曰**：
范丹灰里卜行藏，相对牛衣泣未央；
秋图黄花看冕节，太公八十遇文王。

**释义**：
上天对人都是颇爲公平的，每个人一生中都总会有机会给自己发挥的。故俗语说：人有三衰六旺，旺是比衰多的。但究竟如何掌握这个时机？这个时机究竟有多长久呢？就非人所知了。所以有些人在少年时发迹，甘罗在十二岁拜相，亦有人在晚年发迹，如姜太公八十遇文王。但常心存行善之心，则贵不可言。

# 第92签

**第92签：古人玄德离许都**

**签文**：
```
雷声大，雨点小，纵好不好；
事在解铃人，福自改过，多行善可保。
```

**详解**：

**签文**：
雷声大，雨点小，纵好不好；
事在解铃人，福自改过，多行善可保。

**解曰**：
刘玄德，即刘备，三国时其蜀国之创国君主，受人民爱戴。未称号帝位时，与军师诸葛孔明佔有许都城，后曹操挥军南下，直迫许都城，玄德知己方的军事力量远逊于曹兵，遂离城，并公告欲随其他往者可以一起成行。由于玄德待民宽厚，即号召了十馀万人跟随。求得此签者，凡事只要小心，虽先厄但后得大吉。

**详解**：
谋望
费大力而得益小。
钱财
将有损耗。
婚姻
现在未到时候。
自身
小心行事定卜吉祥。
家宅
正受外来的冲击。
开业
时机未如理想。
迁居
合适，可免灾害。
出行
虽有虚惊，无碍。
疾病
不能即时痊愈。
六甲
一不小心，就会流産。
行人
已在道途中。
诉讼
还是以和解爲宜。

**又曰**：
铁铸风波狱已成，吉人天相得轻从；
更有卓上青蛇绕，诏下金鸡庆自荣。

**释义**：
已经犯下的错误当然无可挽回，但所受的刑责却因有上天的庇荫而得到宽减，所以纵有祸亦非严重。同时，一切问题的症结仍在你的手上，所以要得到解决之方，就要看解铃之人，多行善则可得福。

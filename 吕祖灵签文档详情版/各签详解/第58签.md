# 第58签

**第58签：古人永乐王定番邦**

**签文**：
```
可罢即罢，不须斩尽杀绝，事到无可奈何；
也要守旧安歇，力保无失可得，三七可决。
```

**详解**：

**签文**：
可罢即罢，不须斩尽杀绝，事到无可奈何；
也要守旧安歇，力保无失可得，三七可决。

**解曰**：
永乐乃明成祖年号，爲明太祖朱元璋第四子，后夺取皇位，他解除藩王兵权，并屡次出兵打击蒙古势力，使得番邦不敢扰乱边境。求得此签者，须宅心仁厚，贵乎守成，有寸进便不可冒进，处事待人不可赶尽杀绝，留有一点馀地给人，纵使非不得已，也要不急躁，不可胡乱地妄动，这样可保无所损失。决事当于二十一日内。

**详解**：
谋望
早些办也许成功。
钱财
可以取得。
婚姻
恐防会有突然的变动。
自身
审慎应变可获福。
家宅
宅运又趋平静。
开业
持续努力会成功。
迁居
远近都无妨。
出行
早一点出发较好。
疾病
会很快痊愈。
六甲
非常顺産。
行人
一收到你的通知即来。
诉讼
急急地去进行会胜利。

**又曰**：
三战三北君莫羞，一匡天下霸诸候；
若经沟壑殉小节，盖世功名尽射钩。

**释义**：
很多时做事不是一下子就可以成功的，即使在事前你的准备工夫是如何细心及部署如何慎密，也会因特殊的因素而棋差一着。但只要不惧困难，即使是三战皆北（输了），但只要你一旦成功，每一个人都会因此而对你刮目相看。

# 第78签

**第78签：古人赵子龙救阿斗**

**签文**：
```
牆倾屋拆，大势已裂；
防而无悔，土崩地决。
```

**详解**：

**签文**：
牆倾屋拆，大势已裂；
防而无悔，土崩地决。

**解曰**：
赵子龙爲三国时蜀国一员勇将。当曹操率魏兵大军压境，蜀主刘备妻携子逃进井中避祸，赵在途中与刘妃及阿斗失散，遂折返找寻，于废墟中与刘及斗相遇，刘将斗交付给赵带返予刘备，然后投井身亡，以免拖累。赵凭一枪一剑一马背着阿斗冲破重围得以逃脱。求得此签者，凡事须经重重艰辛始得成功，亦有虽危有救之象。

**详解**：
谋望
略爲迟缓，可以成功。
钱财
托人可以成功，急则不成。
婚姻
要清楚对方的背景方免生出意外。
自身
虽危而有救。
家宅
亦颇安宁。
开业
宜守成，新计划暂停。
迁居
比较适合。
出行
与朋友一起去比较好。
疾病
多会是患肠胃病。
六甲
先难后易。
行人
会迟些到，必至。
诉讼
对方较强，和解爲宜。

**又曰**：
莫嫌袖舞太郎当，横扫千军价自昂。
自古痴人如痴福，羡君终始得安康。

**释义**：
痴人爲何能得痴福呢？此无他，因爲他的「因」是「痴」，所以其得的「果」也是「痴」。这种「痴」不一定是「傻」，可以说是单纯或太专注，所以常得贵人的扶助而得到安康。

# 第75签

**第75签：古人隋杨广夜看琼花**

**签文**：
```
天不佑之，时不合也；
花被风折，人在海底。
```

**详解**：

**签文**：
天不佑之，时不合也；
花被风折，人在海底。

**解曰**：
隋朝君主杨广得悉降琼花，遂驱车前往观尝。此花虽然豔丽，但开花爲时很速，喻好景虽好，但时日无多。求得此签者，做事决策宜小心谨慎，最忌是出门，不管是公干私事皆不宜，爲不吉之兆，亦事情有所阻滞也。若能修德行善，亦能消灾。

**详解**：
谋望
艰以成功，多是昙花一现，便成破败。
钱财
周转困难，小心银行或供应商突然将信贷收缩。
婚姻
会有离异。
自身
修德积福可无灾。
家宅
先荣后枯。
开业
初期不错，如不坚守，随便冒进，易招破败。
迁居
远近都不宜。
出行
防途中有灾殃。
疾病
发病速，需儘快诊治。
六甲
事事小心，注意健康。
行人
到来又匆匆离去。
诉讼
对方和解亦有损克。

**又曰**：
身在高山欲上天，忽生艰阻费周旋；
积劳始信閒爲福，多病方知健是仙。

**释义**：
欲望是无底的深潭，得一想二，得寸进尺，当然这也是人的常情。所以身处高山，就想登天，以爲登天很易，当知道难或因进得退时，就知道事情的真象。因此处理事情有时要从正反二方来看，总予人家留有馀地，亦即留有馀地给自己。

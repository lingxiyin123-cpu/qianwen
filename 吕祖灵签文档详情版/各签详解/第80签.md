# 第80签

**第80签：古人三国纷争**

**签文**：
```
三分而已，何须望奢；
人要回家，只待日西斜。
```

**详解**：

**签文**：
三分而已，何须望奢；
人要回家，只待日西斜。

**解曰**：
三国时代的魏、蜀、吴三国鼎足而立，虽然初段时间大家互相攻伐，但劳而无功。在当时，魏挟天子以令诸侯，地大物博，蜀得汉中之地，有人和而物质富庶，吴得长江天险，人杰地灵，互有优点，故此産生均衡之局面。求得此签者，要贵乎知足，不要得陇而望蜀，否则亦是劳心劳力，终归无大进展。

**详解**：
谋望
难以迅速成功。
钱财
不可贪心，防耗损。
婚姻
虽成，防第三者插手。
自身
知足者常乐。
家宅
无障碍。
开业
宜独资经营，合伙有争拗。
迁居
以南方爲宜。
出行
徒劳无功。
疾病
燥热的食品宜减少。
六甲
孕可成。
行人
未至，有资讯至。
诉讼
表面上协调，提防对立暗中耍手段。

**又曰**：
蜂腰鹤膝，屈而不舒；
见兔顾犬，切勿守株；
失之东隅，收之桑榆。

**释义**：
明明是已有名有利，知道再努力强求也是如此，但亦切勿守株待兔地等待命运的安排，要进取。因爲名利如逆水行舟，不进则退，守成是一个最愚蠢的方法，所以一定要抱着奋斗的精神，这样的话，比纵使失这东隅，也必会收之桑榆的。

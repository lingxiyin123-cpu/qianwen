# 第60签

**第60签：古人武侯伐魏**

**签文**：
```
苦而甘，甘而甘，一子一午。
送喜不送忧，也要自裁生。
```

**详解**：

**签文**：
苦而甘，甘而甘，一子一午。
送喜不送忧，也要自裁生。

**解曰**：
诸葛亮受蜀汉主刘备于白帝城托孤，除辅助阿斗爲蜀后汉主，更预算把魏及吴两国吞併，统一全国。由于魏亦拥有颇多兵力，故此诸葛武侯六出祈山，仍然无功而还，求得此签者，有进又难、退又难的局面，凡事只要坚定心志，则可有小成也。

**详解**：
谋望
要心坚志强，成功不易，而劳心费力，疾病缠身。
钱财
会损耗少许金钱。
婚姻
波折多，迟而不乐观。
自身
修善得平安。
家宅
宅运平平。
开业
坚守自己现在事业更佳。
迁居
那处不比这处好。
出行
去远的地方不宜。
疾病
病症缠身，会较长时间才稍有好转。
六甲
应多检查身体。
行人
在途中受到阻碍。
诉讼
谈判会很长久，双方都有损失。

**又曰**：
衣食从来有定缘，强求强取不能全；
仓箱本是他家物，何必劳心苦求谋。

**释义**：
一个人一生之中有多少衣食，已有定数，所以毋须强自求谋，即使是经过一番苦干后得来的，也会无缘无故的失去，因爲这并非是你的福禄所能享用至的。因此，对于一切强求的物品都应谢绝，这样才能保持自己既有的运势，也不会到最后劳而无功。

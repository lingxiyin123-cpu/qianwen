# 第45签

**第45签：古人李龙寻妻**

**签文**：
```
天时人事，正是好佳期，又恐跷蹊；
但看双入卯金宜，只待十八子一提携。
```

**详解**：

**签文**：
天时人事，正是好佳期，又恐跷蹊；
但看双入卯金宜，只待十八子一提携。

**解曰**：
李龙与妻爲避兵乱，自家乡逃往京城，途中失败。抵京后李龙竭力寻觅，终能破镜重圆。求得此签者，只要凡事不忘其本，则可成就，如果能见善勇爲，则更能获得贵人助力。此提携之人姓李，因「十八子」即「木子」凑合而成的「李」字。

**详解**：
谋望
急则不成，慢慢去办便可。
钱财
富贵由勤而得。
婚姻
三番四次托人则成。
自身
以不忘本爲依归。
家宅
谦和则家兴。
开业
和人合伙会成功。
迁居
远处则不宜。
出行
出外注意回程有阻碍。
疾病
可以治，时间较长。
六甲
安産。
行人
会有消息至。
诉讼
借助他人之力谈判可成。

**又曰**：
富贵由来未许求，羡君骑鹤上扬州；
千般种作皆如意，一路荣封到白头。

**释义**：
古语说：「生死有命，富贵攸关。」所以富贵虽然是每个人所梦想、所祈求，但又未能一一如人所愿可以垂手而得。如果能够常常秉持着满招损、谦受益的心态，贫富也不忘其本，当有奇逢之喜，终归有益。

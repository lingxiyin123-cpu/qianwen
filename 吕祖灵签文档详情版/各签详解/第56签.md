# 第56签

**第56签：古人袁有信卖柴**

**签文**：
```
皇皇典，喜喜贲，天衢平坦；
只待小人得志，休得动风，自然既济。
```

**详解**：

**签文**：
皇皇典，喜喜贲，天衢平坦；
只待小人得志，休得动风，自然既济。

**解曰**：
袁有信爲人忠直，分芥不贪。上山打柴往市场卖，公平交易，爲人所称颂，后遇贵。求得此签者，宜心存忠厚，处世信义爲本，安分守纪，切勿贪求非自己之物。如果命中是有的终须会有的，命中是没有的，任凭如何争取也不能得，故此知足常乐，切勿贪求一时之利而自取其咎。

**详解**：
谋望
可得，但稍迟。
钱财
可得利润，但不多。
婚姻
未到时候。
自身
存忠厚可化凶戾。
家宅
各人安份无咎。
开业
宜守旧业。
迁居
急则有损无益。
出行
延期出发爲佳。
疾病
病势缠绵。
六甲
当心会有难産。
行人
这几日内有消息。
诉讼
以和爲贵，急则必败。

**又曰**：
本来面目自家知，得意还防失意时，逾份要存知足念，贪心全丧旧根基。万事不由人计较，一生都是命安排，命里有时终须有，何劳碌碌苦求谋。

**释义**：
人一生吃多少、穿多少，都由命运所定。有的不会缺少，没有的不会多出，因此要常存知足的念头，然后才可以尽情地享受自己的人生。而尤其最重要的，是得意之时不要太逾份，要提防失意之时又如何；故能存信义立世，则能逢凶化吉。

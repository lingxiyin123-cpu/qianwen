# 第42签

**第42签：古人赤壁鏖兵**

**签文**：
```
只说有愁烦，事到其中兴不阑；
逢午方可良，不必乱张惶。
```

**详解**：

**签文**：
只说有愁烦，事到其中兴不阑；
逢午方可良，不必乱张惶。

**解曰**：
三国时曹操率百万军队南下，孙权大爲震惊；时朝上下臣子主战主和争持不下。刘备爲自己利益，派出孔明入东吴欲联同孙权抗曹。后孔明与孙权大将周瑜取得共职，于赤壁用火攻曹兵，大获全胜。求得此签，目前虽有愁烦，但至午年月日即可解决。

**详解**：
谋望
与人适量有望。
钱财
托人则可成。
婚姻
双方仍在考虑。
自身
镇静勿胡思乱想。
家宅
 宅平安而顺畅。
开业
与人合伙有利。
迁居
以原居不动爲吉。
出行
会得到满意成果。
疾病
虽然疗期较长，无碍。
六甲
顺産，是男孩。
行人
午日必至。
诉讼
托第三者和解会易。

**又曰**：
平生志气与天高，欲临川水钓金鳌；
岂料波深难着力，此身且慢去遨游。

**释义**：
自己怀有满腹才华，当然希望能一展抱负，让世人知道，但时运未至，一方面未爲人所尝识，另方面亦未取得发挥的机会，使得自己的本领毫无着力之处，怎么办？宜守候时机，及再充实自己，必能大器晚成。

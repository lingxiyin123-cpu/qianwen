# 第32签

**第32签：古人郭嘉遗计定辽东**

**签文**：
```
荣华贵，富贵来；
天地生成，多吉少悔。
三九遂心怀，虔许善功福自来。
```

**详解**：

**签文**：
荣华贵，富贵来；
天地生成，多吉少悔。
三九遂心怀，虔许善功福自来。

**解曰**：
三国时郭嘉，字奉孝，爲魏曹操一员将领，爲征服辽东而兴兵，不幸阵亡。临终时遗下计取辽东之策略予曹操，后曹操依计而行，终获成功。求得此签者，目下困厄，寸步难行，必须候灾星退尽，始无妨碍。因此暂不宜鲁莽行事，要知所进退。

**详解**：
谋望
可成，但不能急急成就。
钱财
可得，但须待时。
婚姻
仍未取得双方合意。
自身
发慈善念以灭一切危。
家宅
灾星渐渐退却。
开业
宜先定出较长期策略。
迁居
远方吉。
出行
无妨，途中有贵人助力。
疾病
非常危险，宜速就医。
六甲
须注意产后调养。
行人
迟至，但先有资讯。
诉讼
现阶段有阻，终可获胜诉。

**又曰**：
经营称夷吾，智谋尤子贡；
如觅蝇头利，冬来可称心。

**释义**：
夷吾与子贡均爲春秋时人，以擅长于经商闻名。其中子贡的名声更爲响亮，因他是孔子的学生，善于辞令，经商曹、鲁国之间，富至千金，并常参与政治活动，并受官职于鲁、卫国家。占得此卦，如欲觅取蝇头小利，到冬天时节应可称心如意。

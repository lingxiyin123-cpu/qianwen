# 第37签

**第37签：古人大舜耕田**

**签文**：
```
黄金自重，人天雨映；
春花秋果，何悔何吝。
```

**详解**：

**签文**：
黄金自重，人天雨映；
春花秋果，何悔何吝。

**解曰**：
古时舜帝爲有德行之人，故大尧退位即禅位让予舜帝，他努力耕耘，与民同耕食，故得万民爱载，同心协力，秋天收成时可得大丰收。求得此签者，喻示旦夕有喜，尤其是在秋天时节，做事营谋将渐入佳境及有收益。

**详解**：
谋望
虽可成就，但营谋颇爲艰辛。
钱财
不必过分强求。
婚姻
现仍有少许障碍。
自身
坚持善行以获福。
家宅
家渐上轨道。
开业
会有贵人相助。
迁居
近处无妨。
出行
暂不宜。
疾病
纵病也很容易痊愈。
六甲
顺産，可生子。
行人
早晚即至。
诉讼
以和爲贵，早点解决好。

**又曰**：
随波逐流，瞬息千里；
三鸟飞鸣，一欧翔起。

**释义**：
逆水行舟，是吃力不讨好的，因此人生一直在逆水的环境中的话，是相当辛苦的，这就是人处于逆运的时节了，反过来，如果是顺水的随波逐流，可以在瞬息之间，得走千里，即轻鬆又写意，要达到这个地步，日常积德向善，可减少人生的阻碍。

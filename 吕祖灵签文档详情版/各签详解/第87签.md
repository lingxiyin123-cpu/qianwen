# 第87签

**第87签：古人孔子在陈**

**签文**：
```
锦绣不成文，只恨修之短；
挽回时命，鸡鼠猪牛，
也宜推让，问过收成子罕上。
```

**详解**：

**签文**：
锦绣不成文，只恨修之短；
挽回时命，鸡鼠猪牛，
也宜推让，问过收成子罕上。

**解曰**：
孔子爲春秋时人，被人称爲万世师表，至圣先师，有门上第子三千，具圣贤者七十二人。孔子不独爲一教育家，亦爲一思想家，希望以「仁」治理国家，故曾率领第子周游宋、卫、陈、齐、楚等国，以推广其思想，至陈国时绝粮。其时门下弟子子路怨歎不已，孔子则说，此皆因远不通而有此危。求得此签者，凡做事莫急，待时机来则水到渠成，逢凶化吉之兆。

**详解**：
谋望
最后关头才得。
钱财
现仍处周转不灵阶段。
婚姻
有障碍不成。
自身
失运须修善。
家宅
尚属平安。
开业
有阻力，延迟爲佳。
迁居
不利，一动不如一静。
出行
犹恐有灾厄。
疾病
若长久便有危险。
六甲
略有障碍。
行人
一、两日内即至。
诉讼
向对方陈说利害，可得和解。

**又曰**：
困苦须由命里该，巫医小术亦须才；
守株待兔有时获，缘木求鱼何处来。

**释义**：
当一个人运势不畅通之时候，事事皆有阻滞，君子安穷，要在运蹇时甘之如饴。守株待兔，缘木求鱼，虽然都是千万中不能得一的机会，但仍然是有其一定微细的机遇的。但切勿不能心急，待时至即可。

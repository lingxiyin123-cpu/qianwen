# 第62签

**第62签：古人文昌三圣**

**签文**：
```
吉吉吉，益益益，不必多急，自有文章东壁。
只在初三十一，方可必，方可必。
```

**详解**：

**签文**：
吉吉吉，益益益，不必多急，自有文章东壁。
只在初三十一，方可必，方可必。

**解曰**：
文昌是业管学问的神祗之一，分有文曲星、文星和魁星，主宰功名、禄位，爲读书人所祭祀。因此以前的人说某某文昌或文曲入命或魁星点斗，其必得功名利禄，故文昌爲吉星，初三、十一日爲文昌照顾，自有显赫的文章。求得此签者，可得吉星拱照，虽然途中也有险阻，但终必没有凶险。

**详解**：
谋望
易成。
钱财
托人去，必得成功。
婚姻
极佳，快点办便可。
自身
平安兼发达。
家宅
运势昌隆。
开业
有十分希望。
迁居
无妨。
出行
途中有贵人扶助。
疾病
很快痊愈。
六甲
顺産，生男孩。
行人
即至，与友人同来。
诉讼
进行谈判，并无阻滞。

**又曰**：
如何平地得爲仙，只要功夫不畏难；
吉岁园中表竹莜，今年可作钓鱼竿。

**释义**：
修善修身爲一个人秉持的信念，能够时刻地做到，就可以修炼成神仙，这是修道人所期望的最高境界。但要做到这一步，一定要有下功夫的态度，不畏难的精神，才能够把好的运气儘量地发挥出来。因此已有种福因的人，今岁可收得善果的条件。

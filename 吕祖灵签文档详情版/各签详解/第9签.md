# 第9签

**第9签：古人岳飞出世**

**签文**：
```
鲲化爲鹏一任飞，长安路上好光辉；
阴谋阴卜皆如许，顺水行舟定好归。
```

**详解**：

**签文**：
鲲化爲鹏一任飞，长安路上好光辉；
阴谋阴卜皆如许，顺水行舟定好归。

**解曰**：
岳飞字鹏举，相传出生时有大鹏鸟在其屋顶盘旋。后洪灾暴发，飞坐于木盘上得逃难，求得此签者，凡事先凶后吉，有贵人扶助，险中有救。但必须要行爲正直，安守本份爲妙，若强行其事，反遭凶险。

**详解**：
谋望
正直地去做可以成功。
钱财
从根本上去求取可成功。
婚姻
第二次婚合爲佳。
自身
安守本份以修身。
家宅
坚守自己正直的便平安。
开业
有贵人扶持。
迁居
远近的都适宜。
出行
留意家中有事会折返。
疾病
先危而后安。
六甲
小心产后调理。
行人
已在途中。
诉讼
会很快得到解决。

**又曰**：
远涉波涛一叶舟，如今始得过滩头。
年来心事才成就，屈指从前多可忧。

**释义**：
东奔西走所求的是甚么？积聚多多，很多时亦只不过是爲儿孙作马牛而已！儿孙自然有其福份，只要自己及教导他们积善贻谋便可以，不必有实质的金钱及物业留下来以遗害他们。占者已由当年的忧虑而进展至开始有成就，成功的目标已很接近了。

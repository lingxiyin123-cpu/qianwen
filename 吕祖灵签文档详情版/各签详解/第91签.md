# 第91签

**第91签：古人李旦受劫**

**签文**：
```
莫急走，看绊跌，
云掩风霾，终是一场冤业；
若冤业，修省方免得。
```

**详解**：

**签文**：
莫急走，看绊跌，
云掩风霾，终是一场冤业；
若冤业，修省方免得。

**解曰**：
李旦因受迫害，装疯至陶府拆情，后得马周访主返朝，艰辛之极，然真命天子有百灵护助。求得此签者，命运现正受一时困顿，如舟破加上风狂，当然是危险之极。但由于有第三者的帮助，终能化险爲夷。

**详解**：
谋望
似成而实际不成。
钱财
会有意外之财。
婚姻
强要结合，最终也要离婚。
自身
修善可得平安。
家宅
家运较爲艰辛。
开业
现在尚非其时。
迁居
以不动爲吉。
出行
取消行程爲宜。
疾病
一不小心便会加剧病情。
六甲
要小心注意，防有惊哧。
行人
不会到。
诉讼
防对方不肯和解，否则可速解决。

**又曰**：
尔乘病马渡危坡，难免疏虞失足多；
幸赖执鞭人得力，不教险处有蹉跎。

**释义**：
古时人曾假设最危险的事情是甚么？答桉是：盲人骑瞎马，夜半临深池。试想盲人骑着盲马，深夜在一个深渊的旁边，就如此签乘着病弱之马行至危坡，是很容易因疏忽而失足陷下。犹幸危中有救，有执鞭疆的人辅助，使得临危也有救助。
